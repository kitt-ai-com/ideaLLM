## 현황 진단 (As-Is)

### 있는 것
- Next.js 기반 매거진형 뉴스레터 사이트 (Vercel 배포)
- Issue 10까지 발행된 아티클 아카이브
- 카테고리 4개 (최신 AI / AI 활용법 / 업무 자동화 / 분야별 적용)
- 기본 네비게이션 (아카이브, 최신 호)

### 없는 것 (갭)
- 이메일 구독 폼 → **구독자 수집 불가**
- 서비스 소개 페이지 (교육/컨설팅/AX 개발) → **리드젠 불가**
- CTA 버튼 → **전환 없음**
- 애널리틱스 → **성과 측정 불가**
- 문의 폼 → **인바운드 리드 없음**

## 서비스 소개 페이지 구조 (/services)

### Hero
```
킷AI와 함께 AI를 비즈니스에 적용하세요
[서비스 문의하기 →]
```

### 서비스 카드 3개
1. **AI 교육**
   - 대상: 마케터, 경영진, 실무자
   - 형태: 그룹 강의, 1:1 코칭, 온라인 과정
   - CTA: "교육 문의"

2. **AI 컨설팅**
   - 대상: AI 도입을 검토 중인 기업
   - 형태: 현황 진단 → 전략 수립 → 로드맵
   - CTA: "컨설팅 상담"

3. **AX 개발 (AI 에이전트 구축)**
   - 대상: 업무 자동화, 에이전트 개발 수요 기업
   - 형태: 맞춤 에이전트 설계 및 개발
   - CTA: "개발 문의"

### 케이스스터디 섹션
- 최근 3개 케이스스터디 카드

### Footer CTA
```
궁금한 점이 있으신가요? 카카오톡으로 바로 문의하세요.
[카카오톡 상담] [이메일 문의]
```

## 인라인 CTA 컴포넌트 스펙

아티클 내 2곳 배치:
1. **중간 CTA** (아티클 중간, 자연스러운 맥락):
   ```
   이런 자동화를 우리 회사에 적용하고 싶다면?
   [무료 상담 신청 →]
   ```

2. **하단 CTA** (아티클 끝):
   ```
   킷AI 뉴스레터를 이메일로 받아보세요
   [이메일 주소 입력] [구독하기]
   + 서비스 소개 보기 →
   ```
