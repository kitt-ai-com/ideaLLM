## 1. Opportunity Solution Tree

### 목표 (Outcome)
**뉴스레터를 AI 교육·컨설팅·AX 개발 서비스의 리드젠 엔진으로 전환**
- 현재: 콘텐츠 발행 플랫폼 (독립적)
- 목표: 구독자 → 서비스 문의 전환 퍼널

### 기회 (Opportunities)

| # | 기회 | 중요도 | 현재 해결 |
|---|---|---|---|
| O1 | AI 실무 뉴스를 원하는 SMB 마케터/경영자 | 높음 | 부분적 |
| O2 | AI 교육(강의) 관심 구독자 → 유료 전환 | 높음 | 없음 |
| O3 | AX 개발/컨설팅 문의 유도 | 매우 높음 | 없음 |
| O4 | kitt 브랜드 신뢰도 구축 | 높음 | 부분적 |

### 솔루션 (Solutions)

| 기회 | 솔루션 |
|---|---|
| O1 | 주 1회 AI 뉴스 큐레이션 + 카테고리 필터 강화 |
| O2 | 서비스 소개 페이지 (교육/컨설팅/AX 개발) + CTA |
| O3 | 아티클 내 인라인 CTA + 문의 폼 연동 |
| O4 | 킷AI 케이스스터디 콘텐츠 발행 |

## 2. 페르소나

### Primary: 중소기업 마케터/대표 (AI 전환 검토 중)
- 나이: 35-50
- 고민: "AI 어떻게 써야 하나, 믿을 수 있는 곳에 물어보고 싶다"
- 행동: 뉴스레터 구독 → 사례 읽음 → 문의
- 전환 포인트: "우리 업종에 맞는 사례"를 읽고 연락

### Secondary: AI 관심 직장인 (교육 수요)
- 고민: "실무에서 AI 툴 제대로 배우고 싶다"
- 행동: 아티클 소비 → AI 교육 과정 문의
- 전환 포인트: "무료 체험 강의" CTA

## 4. Beachhead Market

**우선 타겟**: 서울/수도권 SMB 마케터·경영자, AI 도입 검토 단계
- 시장 규모: 국내 중소기업 수 약 70만 개, AI 도입률 ~15% (2025)
- Beachhead: 월 구독자 1,000명 → 서비스 문의 전환율 2-3% 목표

---

## 5. GTM 전략

### Phase 1 (1개월): 기반 구축
- 서비스 소개 페이지 3개 추가 (교육/컨설팅/AX개발)
- 아티클 내 인라인 CTA 삽입
- 카카오톡 오픈채팅 / 폼 문의 연동

### Phase 2 (2-3개월): 성장
- 주 1회 발행 정례화 (Beehiiv or Maily 연동)
- SEO 최적화 (AI 자동화, 업무 자동화 키워드)
- 소셜 배포 (LinkedIn, 인스타그램 카드뉴스)

### Phase 3 (4-6개월): 전환 최적화
- 이메일 시퀀스 (구독 → 소개 → 사례 → CTA)
- 유료 강의 랜딩 페이지
- 케이스스터디 아티클 월 2편

## 7. 핵심 기능 요구사항

### Must Have (MVP)
- [ ] 서비스 소개 페이지: AI 교육, 컨설팅, AX 개발
- [ ] 인라인 CTA 컴포넌트 (아티클 하단 + 중간)
- [ ] 문의 폼 / 카카오 연결

- [ ] 카테고리 구조 강화 (현재 4개 → 6개)

### Nice to Have

- [ ] 북마크/저장 기능
