Chapter 03 · 기능 01

AI 상담 Agent

FEATURE 01

공감하는 AI, 판단은 전문가에게

24시간 첫 접촉을 담당하되, 감정적으로 무거운 상황에서는 즉시 전문 상담사에게 인계합니다.

멀티채널 지원: 카카오채널, 웹 채팅, 음성 상담 지원 — 고객이 편한 방법으로 접촉

서비스 자동 분류: 유품정리 / 특수청소 / 빈집정리 / 고독사 / 사전상담 자동 판별

긴급 감지: "고독사", "충격적", "임종" 등 키워드 → 담당자 즉시 알림 분기

감정 톤 설계: 유족이 쓰는 언어 패턴 학습 → 공감형 응답 템플릿 적용

상담 요약 자동 전달: 대화 내용 구조화 → 관리자 대시보드에 즉시 전송

Chapter 03 · 기능 02

이미지·영상 기반 자동견적 시스템

FEATURE 02

사진 3장으로 10분 안에 견적서 완성

고객이 스마트폰으로 공간을 촬영하면 Vision AI가 분석하고 견적 초안을 자동 생성합니다.

공간 규모 분석: 방 개수, 면적 추정, 물품 밀도 자동 판별

물품 4분류 자동 태깅: 재사용 가능 / 기부 가능 / 재활용 / 폐기 (ESG 연동)

특수 상황 감지: 오염도, 특수청소 필요 여부, 귀중품 예상 위치 알림

표준 단가 적용: 공수 × 물품 카테고리 계수 × 지역 가맹점 단가

담당자 최종 승인: AI 초안 → 담당자 검토 → 고객 발송 (완전 자동화 아닌 AI+Human)

견적 생성 프로세스

1

고객 사진/영상 업로드

2

Vision AI 공간 분석 (30초 이내)

3

물품 자동 분류 + 수량 추정

4

표준 단가 × 계수 견적 초안 생성

5

담당자 검토 (평균 5분)

6

고객 견적서 발송 + 전자계약 안내

Chapter 03 · 화면 구성안

Chapter 04 · 검색 전략

SEO · AEO · GEO 통합 검색 노출 전략

검색은 이미 3가지 전쟁터가 됐습니다. 세 채널 모두에서 선점해야 합니다.

SEO · 검색엔진 최적화

네이버·구글 검색 1위 선점

  • "서울 유품정리 업체 추천" 등 롱테일 100개 키워드 공략
  • "고독사 청소 비용", "부모님 집 정리" 정보성 블로그 월 8편
  • 지역별 랜딩페이지 (서울·경기·부산 각 특화)
  • 네이버 플레이스 최적화, 블로그 체험단 운영
AEO · AI 검색 최적화

ChatGPT·Perplexity 답변에 인용

  • "유품정리 업체 추천해줘" 질문 시 통인이 인용되도록
  • 구조화된 FAQ 페이지 (AI가 읽기 좋은 포맷)
  • 신뢰 지표 다수 배치 (자격증·수상·인증·협회 가입)
  • Wikipedia 스타일 브랜드 정보 페이지 구축
GEO · 생성형 AI 최적화

Perplexity·Gemini 응답에 노출

  • AI가 인용하는 "권위있는 소스"로 포지셔닝
  • 언론 노출 확대 (보도자료 월 1회, AI 도입 앵글)
  • 업계 데이터 레포트 발간 (AI가 참고하는 1차 소스)
  • YouTube 영상 + 자동 자막 → 멀티소스 신뢰도
콘텐츠 마케팅

신뢰를 만드는 콘텐츠

  • 유튜브 숏폼: 작업 전후 비포어앤애프터 (신뢰+바이럴)
  • 고객 사례 스토리 (익명): 가장 강력한 신뢰 도구
  • ESG 실적 카드뉴스: "이번 달 500kg 자원순환"
  • 계절 캠페인: 연말 대청소, 이사철 특가 등

Chapter 05 · 사업화 전략

투자유치 & IPO 대응형 성장 전략

플랫폼 구축은 사업화의 시작입니다. 데이터 자산과 ESG 실적이 투자 가치를 만들고, 프랜차이즈 확장이 IPO 스토리를 완성합니다.

📈 투자유치 스토리

  • 데이터 자산: 국내 유일 엔딩케어 AI 데이터 보유 기업
  • 시장 독점: 전국 브랜드 미개척 → 선점 프리미엄
  • ESG 가치: 자원순환 데이터 → 임팩트 투자 기준 충족
  • 프랜차이즈 멀티플: 가맹점 수 × 평균 매출 × 수수료율
  • 시리즈 A 목표: Phase 2 완료 후 데이터 기반 IR 준비

🏢 프랜차이즈 확장 모델

  • 가맹비 구조: 초기 가맹금 + 월 플랫폼 이용료 + 작업 수수료
  • 표준화 패키지: 교육 프로그램 + 플랫폼 + 브랜드 가이드
  • 2년 목표: 전국 50개 가맹점 → 수익 모델 검증
  • 해외 진출: K-웰다잉 콘텐츠 기반 일본·동남아 진출 검토
  • 협회 연계: 한국엔딩라이프지원협회 자격 체계와 결합

📊 IPO 대응 플랫폼 설계

  • 데이터 리포팅: 투자자 대시보드 — KPI 자동 집계
  • ESG 공시: GRI 기준 자원순환 실적 보고서 자동화
  • 매출 투명성: 가맹점 수수료 + 플랫폼 이용료 구조화
  • 성장 지표: MAU, 건당 매출, 가맹점 수, 재의뢰율 추적
  • 3년 목표: KOSDAQ 상장 또는 M&A Exit 기반 마련

🌏 장기 비전 (2028+)

  • 엔딩케어 산업 표준화 기업으로 포지셔닝
  • AI 플랫폼 기반 B2G 계약 (지자체 복지 인프라)
  • K-Ending Care 글로벌 수출 (고령화 선진국 타겟)
  • 데이터 기반 엔딩케어 정책 컨설팅 사업
  • ESG 임팩트 투자 유치 (사회적가치 측정 가능)
