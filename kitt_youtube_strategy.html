Chapter 02 · 가장 중요한 결정

"페이스리스"는 B2B 신뢰 목표와
충돌한다 — 그래서 우회한다

⚠ 리서치 3개 스트림이 일관되게 경고

· 성공한 한국 B2B 채널(윤자동·조코딩·구씨)은 전부 talking-head + 실제 구축 시연.
· "양산형 AI보이스 페이스리스 = 브랜드화·신뢰 불가" (국내) · "2025.7 유튜브 가이드라인 — 원본성 없는 페이스리스/AI보이스 수익·추천 단속" (해외).
· B2B 의사결정자 75%는 사양보다 '사람의 전문성(사고 리더십)'을 신뢰.

해법 — "진정성 페이스리스(Authentic Faceless)"

얼굴은 안 내보내되, 신뢰는 콘텐츠의 원본성으로 만든다.

신뢰 요소페이스리스로 구현하는 법
사람의 전문성대표(유승범) 본인 목소리 나레이션 권장 — AI보이스보다 신뢰 압도적 ↑. AI보이스 사용 시 단일 브랜드 보이스 고정, 기계적 listicle 금지.
실전 증거화면녹화 대상이 진짜여야 함 — workKITT·sellKITT 실제 대시보드, 실제 고객 빌드 과정, 실측 숫자(87%·90%↓).
차별성(반-양산형)남의 화면·뉴스 읽기 ❌ → 우리가 직접 만든 워크플로우·코드·결과만. 단속 핵심은 '얼굴 없음'이 아니라 '원본성 없음'.
인격핸즈온 시연(손), 일관된 인트로/아웃트로, 케이스 스토리텔링.

권장 진화 경로

0~3개월: 100% 페이스리스(화면녹화 + 본인 목소리). 4개월+: 케이스 스터디·고객 인터뷰 영상에 한해 대표 얼굴/실루엣 일부 노출을 옵션으로 — 신뢰가 곧 리드 전환율이다.

Chapter 07 · 알고리즘 현실

2025.12 대변동 & 신규 채널 0→1

📉 2025.12 홈피드 대변동

롱폼 추천 슬롯 ~80% 축소, 그 자리를 Shorts가 차지. Shorts↔롱폼 랭킹 모델 분리 → 숏폼 성과가 롱폼을 자동으로 올려주지 않음. 연결은 의도적 설계 필요(같은 주제·end screen·고정댓글).

🚀 "구독자 X명 필요"는 미신

구독 0 채널 영상도 노출된다. 신규 영상은 소규모 테스트 → 초기 신호(시청시간+CTR+만족도) 강하면 확장. 2026은 작은 채널을 더 빨리 테스트 + 마이크로 니치 클러스터링 → 좁은 니치 + 높은 만족도가 B2B에 유리.

🔑 태그는 거의 무의미

제목·설명·자막(대사)에 키워드 집중. 잘 랭크된 1개 영상이 월 50~200 리드를 무기한 생성 가능.

👥 첫 100~500 구독은 직접 데려와라

이메일·인스타·링크드인·기존 고객으로 시드 → 알고리즘이 신호 받아 넘겨받음. 첫 10개 영상은 모든 댓글 1시간 내 답글.
