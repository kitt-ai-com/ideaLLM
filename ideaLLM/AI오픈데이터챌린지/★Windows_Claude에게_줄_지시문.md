## 2. 복사해 넣을 지시문 (Windows Claude Desktop 채팅창에)

```
한글에 사업계획서 양식이 열려 있어. (\\Mac\Home\Claude\ideaLLM\ideaLLM\AI오픈데이터챌린지\ 폴더의 사업계획서양식.hwpx, .hwp로 저장해둠)

같은 폴더의 "사업계획서_공식양식_작성초안_킷에이아이.md" 파일을 읽고,
그 내용을 hwp-mcp로 양식의 해당 항목에 채워줘.

작업 규칙:
1. 양식의 항목 제목·번호(1-1, 2-1 등)는 절대 변경/삭제하지 마. 각 항목 아래 본문 영역에만 내용을 채워.
2. 파란색 안내 문구(※로 시작하는 회색/파란 글씨)는 삭제하고, 그 자리에 초안 내용을 검정색으로 작성해.
3. 항목 매핑:
   - [개요(요약)표] → 초안의 "📋 [개요(요약)] 표" 5개 칸
   - 1-1 전반적 개발계획 → 초안 1-1 (◦ 개발배경/목적/핵심기술3원칙/시나리오6단계/차별점)
   - 1-2 개발 추진일정 → 초안 1-2 (10주 로드맵 표)
   - 2-1 AI알고리즘·파운데이션모델 → 초안 2-1 (Claude + 3중방어 표 + 추진체계)
   - 2-2 성능 목표 → 초안 2-2 (정량지표 6개 표)
   - 3-1 현안 해결 → 초안 3-1 (기업/중진공 기대효과 표 + 실증근거)
   - 3-2 경제성·확장 → 초안 3-2 (사회적가치 + 3단계 확장)
   - 4-1 대표자 → 초안 4-1 (유승범 역량)
   - 4-2 팀원 → 초안 4-2 (팀원 현황표 4행)
   - 5-1 수상이력 → 공란 유지 (해당 시만)
   - 5-2 유사 대회/개발이력 → 초안 5-2 (newsletter-agent 등 표)
4. 표는 양식의 표 구조에 맞춰 행 추가하며 채워. 해당없는 표는 공란 유지.
5. 한 항목(1-1, 1-2, 2-1...) 채울 때마다 hwp_get_text로 결과를 보여줘서 내가 확인할 수 있게 해줘.
6. 전부 채운 뒤 "사업계획서_킷에이아이_작성본.hwp"로 저장해줘. (원본 양식은 보존)

표 셀 구조에 민감하니, 한 번에 다 하지 말고 항목별로 채우고 확인하며 진행해줘.
```

## 3. 진행 중 막히면 (대비책)

- **표 채우기가 어긋나면**: 그 표는 hwp-mcp 대신 한글에서 직접 입력 (초안 .md를 보며 수기). 텍스트 항목(1-1, 3-1 등 서술형)은 hwp-mcp가 잘 처리.
- **hwpx 제어 안 되면**: 한글에서 양식 열고 `.hwp`로 저장 후 그 파일로 재시도.
- **도구 이름이 다르면**: 0번에서 확인한 실제 도구명으로 지시문의 hwp_get_text 등을 교체.
