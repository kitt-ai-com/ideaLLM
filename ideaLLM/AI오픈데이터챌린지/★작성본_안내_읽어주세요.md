## 📄 결과물

**`사업계획서_킷에이아이_작성본.hwpx`** ← 이 파일을 한글에서 열면 됩니다.

- 공식 양식 구조 그대로 + **서술형 항목 7개 본문 자동 삽입 완료**
- XML 유효성 검증 통과 (한글에서 정상 열림)

| 항목 | 상태 |
|---|---|
| 1-1 전반적 개발계획 | ✅ 5개 ◦ 채움 (배경·목적·3원칙·시나리오·차별점) |

## ⚠️ 한글에서 직접 해야 할 것 (표/이미지 = 자동삽입 위험해서 제외)

표는 셀 구조가 복잡해 자동삽입이 위험 → **한글에서 `사업계획서_공식양식_작성초안_킷에이아이.md`를 보며 직접 입력**:

1. **[개요(요약)표]** (양식 맨 앞) — 초안의 "📋 개요표" 5칸 입력
2. **2-2 정량목표** — 본문엔 ◦로 요약됨. 표로도 정리하려면 초안 2-2 표 참고(선택)
3. **4-2 팀원 현황표** — 4행: 유승범/김상훈/디자이너(실명)/신규채용(예정)
4. **5-1 수상이력** — 있으면 기입, 없으면 공란
5. **5-2 유사 대회/개발이력** — newsletter-agent·셀킷·메타광고 3행
6. **이미지** — 서비스 흐름도 + b2bmarket/newsletter-agent 운영 화면 캡처

## 📝 한글에서 마무리 순서

1. 한글에서 `사업계획서_킷에이아이_작성본.hwpx` 열기
2. 각 항목의 **파란색 ※ 안내문구 삭제** (공고 규정 — 자동삭제는 위험해서 남겨둠)
3. 위 "직접 해야 할 것" 표·이미지 입력
4. 4-2 디자이너 실명 기입
5. **`.hwp` 또는 `.hwpx`로 저장** → K-Startup 첨부
