# 셀킷(SellKitt) 시스템 구성도

## ■ Mermaid 다이어그램 (mermaid.live 에서 이미지 변환 가능)

```mermaid
flowchart TD
    subgraph USER["👤 수요기업 사용자"]
        A1[웹 브라우저]
        A2[카카오톡\n이미지 주문]
        A3[엑셀 파일\n업로드]
    end

    subgraph SELLKITT["🧠 셀킷 플랫폼 — 킷에이아이 운영 (Vercel)"]
        direction TB
        B1["① AI 엑셀 컬럼\n자동 매핑 API"]
        B2["② 이미지 주문\n자동 추출 API"]
        B3["③ 지능형 배송비\n자동 산정 엔진"]
        B4["④ 공급사 자동\n정산 엔진"]
        B5["⑤ AI 운영 챗봇\n(22개 Function Tool)"]
        B6["⑥ AI 이상탐지\n엔진"]
    end

    subgraph AI["🤖 AI 엔진 (Google)"]
        C1[Gemini 2.5 Flash\nLLM]
        C2[OCR 엔진]
    end

    subgraph DB["🗄️ 데이터베이스 (Railway)"]
        D1[PostgreSQL\n멀티테넌트 격리]
    end

    subgraph NOTIFY["📱 알림"]
        E1[카카오 알림톡\n자동 발송]
    end

    A1 --> SELLKITT
    A2 --> B2
    A3 --> B1
    B1 <--> C1
    B2 <--> C1
    B2 <--> C2
    B5 <--> C1
    SELLKITT <--> D1
    B4 --> E1
    E1 --> A1
```

                 │
