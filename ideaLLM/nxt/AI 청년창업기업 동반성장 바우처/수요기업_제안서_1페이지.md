```

  매일 엑셀로 수작업 취합 중

□ 담당자가 이 반복 업무에 하루 2~4시간 소모
```

## 귀사가 부담하는 것

```
총 사업비: 2억원
  ├ 정부 지원: 1억 7,000만원 (85%) ← NIPA 지급
  └ 귀사 부담: 3,000만원 (15%) ← 현물만 가능 (현금 아님)

현물 = 사무공간·장비·인력 등 귀사가 보유한 자원으로 인정
      → 실제 현금 지출 없음
```

## 셀킷이란?

- **LLM(AI) 기반 B2B 도매·유통 자동화 플랫폼**
- 현재 베타 서비스 운영 중 (b2bmarket.vercel.app)
- 어떤 엑셀 양식이든 AI가 자동 인식 — 기존 업무 방식 변경 불필요
- 카카오톡 이미지 주문 자동 추출
- AI 챗봇으로 자연어 주문 조회·처리
- IT 담당자 없어도 즉시 운영 가능
