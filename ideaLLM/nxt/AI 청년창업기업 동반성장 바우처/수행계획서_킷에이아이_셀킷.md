## I. 개요

### 1. 추진배경 및 목적

#### 가. 목적 및 필요성

국내 B2B 도매·유통 중소기업의 80% 이상은 주문 접수·배송 관리·거래처 정산 업무를
수작업 엑셀로 처리하고 있다. 스마트스토어·쿠팡·카카오톡 등 멀티채널 판매가 보편화되면서
채널별로 상이한 주문 양식을 수작업으로 통합하는 데 담당자 1인당 하루 2~4시간이 소요되며,
오입력으로 인한 오배송·클레임 비율이 전체 주문의 3~8%에 달한다.

이에 킷에이아이(주)는 LLM(대규모 언어모델) 기반 AI 엔진과 자동화 워크플로우를 결합한
B2B 유통 운영 플랫폼 "셀킷(SellKitt)"을 개발·운영 중이다. 본 과제를 통해 셀킷을
제조·유통 분야 수요기업에 공급하여, 주문 처리·배송비 산정·정산 전 과정을 자동화하고
수요기업의 디지털 전환(AX)을 실현한다.

#### 나. 정부지원의 필요성

- 킷에이아이는 2025년 4월 설립된 초기 창업기업으로, 기술력은 검증되었으나
  고객사 레퍼런스 확보를 위한 초기 시장 진입 비용이 최대 장벽임
- 수요기업(제조·유통 중소기업)은 IT 전담 인력이 없어 AI 솔루션 도입 판단·비용 부담이 높음
- 정부 바우처를 통한 매칭 지원으로 양측의 진입 장벽을 동시에 해소하고,
  청년창업기업 초기 판로 형성 및 제조업 AX 가속화를 동시에 달성 가능

### 2. 시장 현황

#### 가. 시장 현황

- 국내 도매·유통업 사업체 수: 약 38만 개 (통계청 2024)
- 이 중 디지털 시스템 미도입 비율: 약 72% (중기부 디지털 전환 실태조사 2024)
- 멀티채널(오픈마켓+SNS+B2B) 동시 운영 기업 비율: 최근 3년간 2.4배 증가
- AI 기반 유통·물류 자동화 솔루션 국내 시장 규모: 2025년 약 4,200억원,
  연평균 28% 성장 전망 (IITP 2025)

#### 나. 시장 동향

- **주요 트렌드**: 단순 ERP에서 AI 에이전트 기반 "무인 운영(Lights-Out Operations)"으로 전환
- **리딩 사업자**: 네이버 커머스, 카카오커머스, 쿠팡 풀필먼트 등 대형 플랫폼은
  자체 AI 물류 자동화 구축. 중소 도매상에 특화된 All-in-One SaaS는 공급 공백 상태
- **문제점**: 대형 플랫폼 솔루션은 중소 도매상 업무 흐름(공급사 수기 발주, 셀러 수기 주문,
  카카오톡 기반 소통)과 미스매치 — 킷에이아이의 기회 영역
- **기회**: LLM의 비정형 데이터 처리 능력이 "어떤 엑셀 양식이든 자동 인식"을 가능하게 하여
  기존 솔루션의 가장 큰 한계(포맷 표준화 요구)를 제거

## III. 세부 추진 계획

### 1. 사업 개요

#### 가. 비즈니스 모델

```
[수요기업 — 제조·유통 중소기업]
   다수 판매채널(스마트스토어·쿠팡·카카오톡·자사몰)에서 주문 수신
         ↓ 기존: 채널별 엑셀 수작업 취합 (1~4시간/일)
         ↓ 셀킷 도입 후: AI가 어떤 양식이든 자동 인식·통합 (5분 이내)
   
[셀킷 AI 엔진 — 킷에이아이 제공]
   ① AI 엑셀 컬럼 자동 매핑 (LLM + 규칙 기반)
   ② 이미지 주문 자동 추출 (OCR + LLM)
   ③ 지능형 배송비 자동 산정
   ④ 공급사 자동 발주 → 송장 자동 매칭
   ⑤ 정산 자동 집계 + 카카오 알림톡 자동 발송
   ⑥ AI 운영 챗봇 (자연어 업무 처리)
         ↓
[수요기업 운영 효과]
   주문 처리 시간 90% 절감 / 오류율 87% 감소 / 정산 업무 자동화
```

**수익 구조 (킷에이아이)**
- 초기 구축비: AI 솔루션 도입·커스터마이징 (바우처 해당)
- MRR 전환: 월 구독형 SaaS 전환 (바우처 종료 후)

#### 나. 사업 추진 전략 (AI 활용 전략)

**전략 1 — 제로 코딩 AI 도입**: 수요기업의 기존 업무 프로세스(엑셀, 카카오톡 주문)를
변경하지 않고 AI가 기존 데이터를 그대로 인식·처리. IT 담당자 없이도 즉시 사용 가능.

**전략 2 — 이중 안전망 구조**: LLM이 1차 처리 → 불확실한 항목은 confidence 점수로
자동 표시 → 운영자 검토 유도 → 규칙 기반 엔진이 폴백 처리. AI 오작동 시에도
업무가 중단되지 않는 무중단 자동화.

**전략 3 — 멀티테넌트 SaaS 확장**: 수요기업 1개사 도입 후 동일 플랫폼에서
추가 기업 온보딩. 추가 개발 없이 신규 고객 즉시 수용 가능한 확장 구조.

#### 다. 사업의 혁신성

기존 유통 관리 솔루션(ERP/WMS)과 셀킷의 핵심 차별점:

| 구분 | 기존 ERP/WMS | 셀킷(SellKitt) |
|---|---|---|
| 엑셀 업로드 | 특정 템플릿 형식만 허용 | **어떤 양식이든 AI가 자동 인식** |
| 이미지 주문 처리 | 불가 | **카카오톡 캡처 이미지도 자동 추출** |
| 배송비 계산 | 단순 고정/무료 기준 | **수량·무게·합배송·반복규칙 자동 판단** |
| 운영 방식 | 화면 조작 필수 | **AI 채팅으로 자연어 업무 처리** |
| 이상탐지 | 없음 | **주문급증·반품률·재고소진 AI 자동 감지** |
| 정산 | 엑셀 수작업 | **공급사별 자동 집계 + 알림톡 자동 발송** |

### 2. 사업 추진 계획 및 방법

#### 가. 사업 추진에 필요한 AI 솔루션

**솔루션명: 셀킷(SellKitt) — B2B 유통 AI 운영 자동화 플랫폼**

| 모듈 | AI 기술 | 주요 기능 |
|---|---|---|
| AI 엑셀 매핑 엔진 | LLM (Gemini 2.5 Flash) + 규칙 기반 | 다채널 주문 엑셀 컬럼 자동 인식·매핑 |
| 이미지 주문 추출 | OCR + LLM | 카카오톡·쿠팡 화면 캡처에서 주문 자동 입력 |
| 지능형 배송비 엔진 | 규칙 추론 알고리즘 | 수량별·무게별·합배송·반복규칙 자동 산정 |
| 운영 AI 챗봇 | LLM + Function Calling (22개 도구) | 자연어로 주문조회·상태변경·통계 처리 |
| 이상탐지 엔진 | 통계적 이상 감지 | 주문급증/급감·반품률·재고소진 실시간 경보 |
| 정산 자동화 | 순수함수 기반 계산 엔진 | 매출·수수료·클레임 차감·VAT 자동 집계 |

**해당 AI 솔루션이 본 과제에 적합한 이유**

수요기업(제조·유통 중소기업)의 가장 큰 AI 도입 장벽은
"우리 엑셀 양식이 워낙 다양해서 시스템에 맞추기 어렵다"는 것이다.
셀킷의 LLM 기반 엑셀 매핑 엔진은 이 문제를 근본적으로 해결한다.
수요기업이 기존에 사용하던 어떤 양식의 주문서도 AI가 자동으로 인식·변환하므로,
기존 업무 방식을 변경할 필요가 없다.

**솔루션 적용 방식**: 구독형 SaaS (초기 구축비 + 월 정액)
**과금 방식**: 초기 셋업 + 월 구독 (MRR 전환)
**AI 솔루션 제공 방식**: 클라우드 서비스 (수요기업 설치 불필요, 웹 브라우저 접속)

#### 나. AI 솔루션 적용(구축) 계획

**1단계: 수요기업 환경 분석 (7월 1~2주)**
- 현재 사용 중인 판매채널 목록 파악 (스마트스토어·쿠팡·카카오톡 등)
- 현재 주문 처리 프로세스 워크플로우 맵핑
- 사용 중인 엑셀 양식 샘플 수집 (3~10종)
- 공급사 수·정산 주기·배송정책 현황 파악

**2단계: 셀킷 테넌트 설정 및 데이터 마이그레이션 (7월 2~3주)**
- 수요기업 전용 테넌트 생성 (멀티테넌트 격리)
- 상품 DB 등록 (엑셀 대량 업로드 또는 수동 등록)
- 공급사 정보 등록 및 정산 주기 설정
- 배송정책 템플릿 구성 (수요기업 현행 배송비 체계 반영)
- 셀러 계정 생성 및 초대링크 발송

**3단계: AI 엑셀 매핑 커스터마이징 (7월 3주~8월 1주)**
- 수요기업 실제 주문 엑셀 양식으로 AI 매핑 테스트
- LLM confidence 임계값 조정 (수요기업 양식 특성 반영)
- 오매핑 케이스 수집 → 규칙 기반 폴백 로직 보강
- 상품 alias 등록 (채널별 상품명·옵션명 매핑 테이블 구성)

**4단계: 파일럿 운영 및 검증 (8월~9월)**
- 소규모 실주문 데이터로 파일럿 운영 (2~4주)

수요기업이 도입 후 일상 업무에서 셀킷을 활용하는 시나리오:

**[시나리오 1 — 아침 주문 처리 (기존 2시간 → 15분)**
```
1. 스마트스토어·쿠팡·카카오톡에서 전날 주문 데이터 수집
2. 셀킷 "AI 엑셀 업로드" 버튼 → 채널별 엑셀 파일 업로드
3. AI가 컬럼 자동 매핑 (confidence 표시)
4. 불확실 항목만 운영자 확인 후 [확인] 클릭
5. 전체 주문 일괄 등록 완료 → 공급사 발주 자동 생성
```

**[시나리오 2 — 카카오톡 주문 처리 (기존 수기입력 5분/건 → 30초/건)**
```
1. 고객이 카카오톡으로 주문 캡처 이미지 전송
2. 셀킷 "이미지로 입력" → 이미지 업로드
3. OCR+LLM이 수령자명·전화·주소·상품·수량 자동 추출
4. 추출 결과 확인 후 [주문 생성] 클릭
```

**[시나리오 3 — 정산 처리 (기존 주 4시간 → 실시간)**
```
1. 셀킷 정산 관리 → 정산 예정 탭 실시간 확인
2. 정산주기 도래 → 자동으로 집계 완료 상태
3. 공급사 지급 확인 후 [입금 완료] 클릭
4. 카카오 알림톡 자동 발송 → 공급사에게 정산 완료 통보
```

```
AI챗봇: "ORD-260601-0023 주문 상태를 배송중으로 변경했습니다."
```

#### 라. AI 솔루션 도입에 따른 혁신성

**혁신 포인트 1 — "어떤 양식이든" 자동 인식**
기존 모든 유통 관리 시스템은 특정 포맷의 엑셀 템플릿을 요구한다.
셀킷은 Gemini 2.5 Flash LLM이 컬럼 헤더와 샘플 데이터를 동시에 분석하여
어떤 양식이든 자동으로 표준 필드에 매핑한다. confidence 점수로 불확실한
매핑을 자동 표시하고, LLM 실패 시 규칙 기반 엔진이 즉시 폴백하는
이중 안전망 구조로 업무 연속성을 보장한다.

**혁신 포인트 2 — 이상탐지 기반 선제적 운영**
단순 업무 자동화를 넘어, AI 이상탐지 엔진이 운영 데이터를 실시간 분석한다.
주문 급증/급감(±70% 임계값), 반품률 이상, 재고 소진, 7일 이상 미수신 발주 등을
자동 감지하여 관리자에게 경보를 발송한다. 문제가 커지기 전에 선제적으로 대응
가능한 "AI 운영 보조" 역할을 수행한다.

**혁신 포인트 3 — 자연어 업무 처리 (AI 챗봇 22개 도구)**
별도 교육 없이 자연어로 "오늘 주문 현황", "홍길동 주문 배송중으로 바꿔줘",
"이번 달 매출 알려줘" 등을 처리할 수 있다. 주문조회·상태변경·상품검색·
매출통계·클레임조회 등 22개 업무 도구를 AI가 자동 선택·실행한다.
