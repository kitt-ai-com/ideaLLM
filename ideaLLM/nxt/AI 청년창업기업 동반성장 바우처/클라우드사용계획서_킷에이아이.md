## 1. 클라우드 서비스 활용 개요

셀킷(SellKitt) AI 솔루션은 온프레미스 설치 없이 **순수 클라우드 기반 SaaS** 형태로 제공됩니다.
수요기업은 웹 브라우저 접속만으로 즉시 서비스를 이용할 수 있으며, 별도의 서버·소프트웨어 설치가 필요 없습니다.

**클라우드 서비스 유형**: IaaS + PaaS + SaaS 혼합
**클라우드 제공사**: Vercel (미국), Railway (미국), Google Cloud (Gemini AI API)

---

## 2. 클라우드 서비스 구성 상세

### 가. Vercel — 프론트엔드 및 API 서버

| 항목 | 내용 |
|---|---|
| 서비스 유형 | PaaS (Platform as a Service) |
| 사용 용도 | Next.js 16 앱 서버 (프론트엔드 + API Routes) |
| 데이터센터 위치 | 미국 (us-east-1), 글로벌 CDN |
| 사용 요금제 | Pro Plan ($20/월 + 사용량 추가) |
| 주요 기능 | 서버리스 함수 실행, 글로벌 CDN, HTTPS 자동 적용, 자동 배포 |
| 월 예상 비용 | 약 98,000원 (기본 $20 + 추가 트래픽) |

**활용 내역:**
- AI 엑셀 컬럼 자동 매핑 API (Gemini API 호출)
- 이미지 주문 자동 추출 API (OCR + LLM)
- 운영 AI 챗봇 API (22개 Function Calling Tool)
- 주문·상품·정산·배송비 관리 API
- 카카오 알림톡 자동 발송 API

---

### 나. Railway — 백엔드 서버 및 PostgreSQL 데이터베이스

| 항목 | 내용 |
|---|---|
| 서비스 유형 | PaaS + DBaaS |
| 사용 용도 | Node.js 서버 + PostgreSQL 16 데이터베이스 |
| 데이터센터 위치 | 미국 (us-west-1) |
| 사용 요금제 | Hobby Plan 이상 ($5/월 기본 + 사용량 기반) |
| 주요 기능 | 자동 스케일링, SSL/TLS 암호화, 자동 백업 |
| 월 예상 비용 | 약 100,000원 (서버 + DB 합산) |

**활용 내역:**
- 멀티테넌트 데이터 저장 (테넌트별 격리)
- 주문·상품·정산·셀러 데이터 CRUD
- Prisma ORM 기반 마이그레이션 관리
- 감사 로그 기록 (SUPER_ADMIN 전용)

---

### 다. Google Gemini API — AI 처리 엔진

| 항목 | 내용 |
|---|---|
| 서비스 유형 | AI/ML API (SaaS) |
| 사용 모델 | Gemini 2.5 Flash (최신 경량 고성능 LLM) |
| 사용 용도 | 엑셀 컬럼 매핑, 이미지 주문 추출, AI 챗봇 Function Calling |
| 가격 구조 | 입력 $0.075/1M tokens + 출력 $0.30/1M tokens |
| 월 예상 사용량 | 입력 약 1,500만 토큰 + 출력 약 50만 토큰 |
| 월 예상 비용 | 약 500,000원 |

**활용 내역:**

| 기능 | 처리 방식 | 예상 토큰/건 |
|---|---|---|
| AI 엑셀 컬럼 자동 매핑 | 컬럼 헤더 + 샘플 데이터 → 표준 필드 매핑 | 입력 3,000 / 출력 500 |
| 이미지 주문 추출 | 이미지 → 주문 정보 JSON 추출 | 입력 5,000 / 출력 800 |
| AI 운영 챗봇 | 자연어 쿼리 → Function Calling → 응답 | 입력 2,000 / 출력 300 |
| 이상탐지 설명 | 감지 결과 → 자연어 설명 생성 | 입력 1,500 / 출력 200 |

## 4. 클라우드 보안 및 데이터 보호

### 가. 보안 조치

| 항목 | 조치 내용 |
|---|---|
| 전송 암호화 | TLS 1.3 (HTTPS 전 구간 강제 적용) |
| 정적 보안 헤더 | CSP, HSTS, X-Frame-Options, X-Content-Type-Options |
| 데이터베이스 암호화 | Railway PostgreSQL SSL 연결 필수 |
| API 인증 | JWT 토큰 기반 (Bearer 방식, 만료 설정) |
| 멀티테넌트 격리 | tenantId 필터 — 타 기업 데이터 접근 물리적 차단 |

### 나. 가용성 및 복구

| 항목 | 수준 |
|---|---|
