1. 제안 개요

1.1 제안 배경 및 목적

고객의 비즈니스 과제

슬리밍고는 15년간 축적된 자세/체형 관리 노하우와 전국 11개 지점이라는 강력한 자산을 보유하고 있습니다. 그러나 이 자산들이 하나로 연결되지 않아, 비즈니스 성장의 발목을 잡고 있습니다.

현재 과제비즈니스 영향해결 방안
지점별 고객 정보 분산 — 11개 지점이 각각 고객을 관리중복 등록, 지점 이동 고객 이탈, 전사 현황 파악 불가모든 지점 고객을 하나의 시스템에 통합
고객 여정 단절 — 상담, 프로그램, 측정, 제품 정보가 따로 관리됨맞춤 케어 불가, 재구매 시점 놓침전 과정을 타임라인으로 시각화
실시간 현황 파악 불가 — 본사에서 전 지점 현황을 즉시 확인할 수 없음느린 의사결정, 지점 비교 불가본사 실시간 대시보드 제공
고객 관리 사각지대 — 재방문 시기를 놓쳐 고객이 조용히 떠남고객 유지율 저하자동 리마인더로 고객 관리

프로젝트 목적

  1. 고객 정보 통합: 11개 지점의 신규 고객 데이터를 하나의 시스템으로 통합
  2. 고객 여정 시각화: 첫 방문 → 상담 → 프로그램 구매 → 재방문 → 제품 구매 → 지속 방문까지 전체 여정을 한눈에 보고 관리
  3. 운영 현황 파악: 본사에서 전 지점의 고객 유치, 매출, 고객 진행 현황을 실시간 확인
  4. 업무 효율화: 반복 업무를 시스템화하여 직원의 업무 부담 절감

기대 목표

목표기대 효과
고객 유지율 향상재방문율 20% 이상 향상
업무 시간 절감고객 정보 조회/입력 시간 50% 이상 단축
매출 가시성 확보본사에서 전 지점 실시간 조회 가능
고객 데이터 오류 감소통합 관리로 중복/누락 90% 이상 감소

1.3 핵심 가치 제안

"11개 지점의 고객을 하나로 연결하여, 첫 방문부터 지속 방문까지 전체 여정을 한눈에 관리하는 시스템을 구축합니다."

고객 여정 관리 — 핵심 기능

2. 현황 분석 및 요구사항

2.1 기능 목록

총 34개 기능 (7개 모듈 29개 + 공통 5개)

A. 고객 관리 (9개)

기능명설명
고객 등록신규 고객 정보 등록 (이름, 연락처, 생년월일, 성별, 주소 등)
고객 프로필 조회/수정고객 상세 정보 조회 및 수정
고객 검색이름, 연락처, 회원번호로 빠른 검색
고객 목록 조회지점별/전체 고객 목록 조회 (필터, 정렬)
엑셀 일괄 등록엑셀 파일로 다수 고객 한꺼번에 등록
고객 여정 타임라인전체 여정을 한 화면에서 확인, 드래그로 단계 이동
고객 태그/메모고객별 태그 부여 및 메모 기록
고객 이동(끌어오기)타 지점 고객을 우리 지점으로 끌어오기 (담당 지점 변경)
재방문 알림/리마인더마지막 방문 후 일정 기간 경과 시 자동 알림

B. 자세측정 데이터 관리 (3개)

기능명설명
측정 데이터 등록자세측정 결과 데이터 수동 입력
측정 이력 조회고객별 과거 측정 데이터 목록 및 상세 조회
before/after 비교두 시점의 측정 데이터를 나란히 비교하여 변화 확인

C. 프로그램 관리 (3개)

기능명설명

고객 프로그램 등록고객에게 프로그램 배정 (종류, 시작일, 회차 등)
프로그램 진행 관리회차별 진행 상태 기록 (완료/미완료/취소)

D. 상담 관리 (3개)

기능명설명
상담 기록 등록상담 내용, 유형, 담당자, 일시 기록
상담 이력 조회고객별 전체 상담 이력 조회
상담 템플릿자주 쓰는 상담 내용 템플릿으로 빠르게 입력

E. 매출 관리 (3개)

기능명설명
구매 기록 등록고객이 구매한 프로그램/제품과 금액 기록
구매 이력 조회고객별 구매 내역 및 총 구매 금액 조회
매출 현황지점별/월별 매출 요약 조회

F. 본사 대시보드 (2개)

기능명설명
지점별 통계지점별 고객 유치 수, 매출, 고객 단계별 진행 현황
전체 현황 요약전 지점 합산 고객 수, 매출, 신규/재방문 고객 추이

기능명설명

권한 관리역할별 접근 권한 설정 (본사관리자/지점관리자/지점직원)
지점 관리지점 정보 등록/수정
시스템 설정알림 설정, 기본값 관리
데이터 백업/복원정기 데이터 백업 및 복원
접속 로그사용자 로그인/주요 활동 로그 기록

공통 모듈 (5개)

로그인/로그아웃, 비밀번호 재설정, 역할 기반 접근 제어, 공통 레이아웃, 공통 화면 요소

3. 주요 사용자 흐름

신규 고객 등록 → 측정 → 프로그램 배정

신규 고객 정보 입력 → 자세측정 결과 등록 → 프로그램 배정 → 구매 금액 기록 → 고객 여정 타임라인에서 전체 이력 확인

재방문 고객 관리

고객 검색 → 이전 이력 확인 → 이번 방문 측정 → before/after 변화 확인 → 상담/프로그램 기록

본사 운영 현황 파악

대시보드에서 전 지점 현황 확인 → 지점별 고객 유치, 매출, 고객 단계별 비교

고객 지점 이동 (끌어오기)

타 지점 고객 검색 → "끌어오기" 요청 → 담당 지점 변경 → 이전 이력 포함 전체 확인
