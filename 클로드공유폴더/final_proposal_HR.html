1. 제안 개요

1.1 현재 상황

슬리밍고는 자세/체형 관리 전문 브랜드로서 전국 11개 지점과 본사를 운영하며 빠르게 성장하고 있습니다. 첫방문이벤트(55,000원)부터 쓰리데이(500,000원)까지 6종의 프로그램과 자세 교정 의류 판매/렌탈 사업을 전개하고 있으며, 각 매장에서 활발한 고객 상담과 서비스를 제공하고 있습니다.

그러나 현재 인사관리 영역에서는 별도의 시스템 없이 엑셀 업무일지만으로 모든 것을 관리하고 있어, 아래와 같은 실질적인 문제들이 발생하고 있습니다.

영역현재 방식실제 발생하는 문제
출퇴근 관리엑셀에 수기 기록기록 누락이 잦고, 근무시간 산정이 부정확합니다. 초과근무나 지각 여부를 사후에 확인하기 어렵습니다
직원 성과 평가관리자의 기억과 감에 의존프로그램별 매출, 고객 수, 리뷰 점수 등이 체계적으로 집계되지 않아 공정한 평가가 어렵습니다
본사 직원 평가별도 기준 없음마케팅, 교육, 운영 등 역할별 성과를 측정할 기준 자체가 없습니다
실적 취합매달 각 지점에서 수동 취합11개 지점의 실적을 취합하는 데만 상당한 시간이 소요되고, 수치 오류가 발생합니다
인사 현황 파악파편화된 엑셀 파일전체 조직의 인력 현황, 직원 이력을 한눈에 파악하기 어렵습니다

1.2 프로젝트 목적

본 프로젝트는 슬리밍고의 인사관리를 엑셀 기반에서 웹 기반 통합 시스템으로 전환하여, 다음 네 가지 핵심 목적을 달성하고자 합니다.

1) 정확한 근태 관리

  • 회사 사원카드를 발급하고, 각 지점 바코드 리더기에 스캔하여 출퇴근을 체크합니다
  • 근무시간, 초과근무, 지각/조퇴가 자동으로 계산됩니다
  • 사원카드 한 장으로 어느 지점에서든 출퇴근이 가능합니다

2) 공정하고 투명한 성과 평가

  • 매장 직원은 프로그램 매출, 제품 매출, 고객 수, 리뷰 점수, 상담 건수 등 객관적 숫자로 평가합니다
  • 본사 직원은 각 역할에 맞는 개별 KPI를 설정하여 평가합니다
  • 월별 실적이 자동 집계되고, 반기/연간 종합 평가까지 시스템에서 일관되게 관리합니다

3) 업무일지와 KPI의 자동 연결 (AI 기반) -- 핵심 차별 기능

  • 직원이 매일 작성하는 업무일지를 AI가 분석하여 KPI 항목과 자동으로 매칭합니다
  • "오늘 3건의 프로그램 상담을 진행했습니다"라고 쓰면, AI가 상담 건수 KPI에 자동 반영합니다
  • 현재 달성률과 함께 구체적인 개선 방향까지 피드백합니다

4) 데이터 기반 경영 의사결정

  • 본사에서 전국 11개 지점의 인력 현황, 성과, 근태 상황을 실시간으로 파악합니다
  • 지점 간 성과 비교, 직원별 성과 추이 등을 대시보드에서 한눈에 확인합니다

3. 제안 솔루션

3.1 핵심 차별점 3가지

기존 HR 솔루션은 일반적인 근태 관리와 인사 관리에는 적합하지만, 슬리밍고의 비즈니스에 맞는 "매장별 프로그램/제품 매출 기반 KPI 평가"를 지원하지 못합니다. 본 시스템은 슬리밍고만을 위한 세 가지 핵심 차별점을 갖추고 있습니다.

차별점 1: AI 업무일지 → KPI 자동 연결 (가장 큰 차별 기능)

"업무일지를 쓰면, AI가 알아서 성과에 반영해 줍니다"

기존 방식에서는 직원이 업무일지를 작성하고, 관리자가 이를 별도로 확인한 뒤, 다시 성과 평가 시트에 수기로 옮겨야 했습니다. 이 과정에서 누락과 오류가 발생하고, 관리자의 시간이 낭비됩니다.

본 시스템에서는 이 과정을 AI가 자동으로 처리합니다.

작동 방식:

  1. 직원이 업무일지를 자유롭게 작성합니다.
    예: "오늘 첫방문이벤트 고객 2명 상담, 1명 등록 완료. 스마트 프로그램 기존 고객 1명 추가 등록. 네이버 리뷰 요청 드렸고 1건 등록 확인."
  2. AI가 자동으로 분석하여 KPI에 매칭합니다.
    • 상담 건수: +2건 (첫방문이벤트 고객 2명 상담)
    • 프로그램 매출: +55,000원 (첫방문이벤트 1건)
    • 프로그램 매출: +110,000원 (스마트 1건)
    • 리뷰 점수: +1건 (네이버 리뷰 1건 확인)
  3. AI가 달성률 피드백 + 개선사항을 제안합니다.
    • 이번 달 프로그램 매출 달성률 68%, 리뷰 달성률 60% 등
    • "리뷰 달성률이 60%로 가장 낮습니다. 이번 주 남은 4일간 하루 1건씩 리뷰를 요청하시면 목표 달성이 가능합니다."

이 기능이 가져오는 가치:

기존 방식AI 자동 연결 후
업무일지와 성과 평가가 별개 → 관리자가 수기 대조업무일지 작성 = 성과 기록 (자동 연결)
월말에야 달성률 확인 가능매일 실시간으로 달성률 확인
"뭘 더 해야 하는지" 스스로 판단AI가 구체적인 개선 방향 제시
관리자가 개별 피드백에 시간 소요AI가 1차 피드백, 관리자는 핵심만 보충

차별점 2: 슬리밍고 맞춤형 매장 KPI 평가 체계

"프로그램 6종의 매출을 자동으로 구분하고, 5대 KPI로 공정하게 평가합니다"

기존 HR 솔루션에서는 "전체 매출"만 집계할 수 있지만, 슬리밍고에서는 프로그램별 매출 구분이 필수적입니다.

슬리밍고 매장 직원 5대 KPI:

KPI 항목측정 방식관리자가 설정 가능한 것
프로그램 매출첫방문이벤트, 원데이, 쓰리데이, 스마트, 셀톡신, 에너지밸런스 -- 프로그램별 매출 자동 분류월간 목표 금액, 프로그램별 가중치
제품 매출자세 교정 의류 판매/렌탈 매출월간 목표 금액
고객 수담당 고객 수, 신규 고객 유치 수월간 목표 고객 수
리뷰 점수고객이 남긴 매장/직원 평가 점수월간 목표 리뷰 건수, 최소 점수 기준
상담 건수고객 상담 건수 (전화/방문/온라인)월간 목표 건수

각 KPI 항목의 비중(가중치)을 본사에서 자유롭게 설정할 수 있어, "이번 분기는 신규 고객 확보에 집중하자"는 전략적 방향을 평가 체계에 즉시 반영할 수 있습니다.

차별점 3: 11개 지점 통합 실시간 성과 관리

"본사에서 전 지점의 상황을 한 화면에서 실시간으로 파악합니다"

11개 지점이 각각의 엑셀 파일로 관리되던 방식에서, 하나의 통합 대시보드로 전환됩니다.

본사 통합 대시보드에서 확인할 수 있는 것:

  • 인력 현황: 전체 직원 수, 지점별 인원 배치, 신규 입사/퇴사 현황
  • 근태 현황: 오늘 전 지점 출근율, 지각/결근 현황, 초과근무 발생 지점
  • 성과 현황: 지점별 프로그램 매출 순위, 제품 매출 순위, 이번 달 목표 달성률
  • 평가 현황: 진행 중인 평가 완료율, 미제출 직원 현황

지점 간 성과 비교 기능으로 건전한 경쟁을 유도하고, 개인별 성과 추이 그래프로 직원의 성장 흐름을 한눈에 파악할 수 있습니다.

업무일지 접근 제한본인의 업무일지만 조회 가능, 관리자는 소속 지점 직원 것만 조회
접근 기록인사정보 조회/수정/삭제 시 자동 기록

4. UI/UX 설계 방향

4.1 UX 설계 원칙

슬리밍고 인사관리 시스템의 주요 사용자는 IT 전문가가 아닌 매장 직원과 관리자입니다. 따라서 다음 5가지 원칙을 UX 설계의 기본 방향으로 설정합니다.

원칙핵심 내용
3초 안에 이해되는 화면화면을 열었을 때 3초 안에 "이 화면에서 뭘 할 수 있는지" 파악할 수 있어야 합니다. 전문 용어 대신 일상 언어를 사용합니다
2번 이내 클릭으로 핵심 기능 도달가장 자주 쓰는 기능(출퇴근 체크, 내 실적 확인)은 로그인 직후 첫 화면에서 바로 사용할 수 있습니다
실수해도 되돌릴 수 있는 안전함중요한 작업에는 확인 단계를 두고, 잘못 입력한 기록은 수정 요청으로 보정할 수 있습니다
PC와 스마트폰 모두에서 편안한 사용매장 직원은 스마트폰, 본사 관리자는 PC를 주로 사용하므로 반응형 디자인을 적용합니다
데이터가 스스로 말하게 하는 시각화숫자 나열 대신 그래프, 게이지, 진행률 바 등 시각적 요소를 적극 활용합니다

4.2 핵심 사용자 시나리오

시나리오 1: 매장 직원의 하루 -- 출퇴근 + 업무일지 + AI 피드백

사용자: 강남점 매장 직원 김수진 (입사 6개월)

아침 출근 (오전 9:50)

  1. 매장에 도착하여 바코드 리더기에 사원카드를 스캔합니다
  2. 시스템에 09:50 출근이 자동으로 기록됩니다
  3. 시스템에 접속하면 이번 달 실적 요약을 확인할 수 있습니다

업무 중 (오후 5시) -- AI 업무일지 작성

  1. 오늘 업무를 마무리하며 업무일지를 작성합니다
  2. 자유롭게 텍스트로 입력합니다
  3. "저장 및 AI 분석" 버튼을 누르면 AI가 업무일지를 분석합니다
  4. AI 분석 결과(KPI 자동 매칭), 달성률 업데이트, 개선 제안이 화면에 표시됩니다
  5. 김수진 직원은 AI 피드백을 확인하고, 내일 리뷰 요청에 더 신경 쓰기로 합니다

퇴근 (오후 6:10)

  1. 퇴근 시 바코드 리더기에 사원카드를 스캔합니다
  2. 18:10 퇴근이 기록되고, 오늘 근무시간이 자동 계산됩니다

시나리오 2: 매장 관리자의 팀원 성과 확인 + 평가

사용자: 강남점 점장 이현우

  1. PC 브라우저에서 시스템 접속 → 지점 대시보드가 첫 화면
  2. 오늘 출근 현황, 이번 달 매출, 목표 달성률을 한눈에 확인
  3. 직원별 KPI 현황 테이블에서 각 직원의 5대 KPI 달성률 확인
  4. 반기 평가 시 시스템이 자동 집계한 6개월 KPI 실적을 기반으로 상위자 평가 작성

시나리오 3: 본사에서 전 지점 성과 비교 대시보드 조회

사용자: 슬리밍고 본사 운영팀장 박정민

  1. 본사 통합 대시보드에서 전 직원 인력 현황, 출근율, 총 매출, 달성률 확인
  2. 지점별 프로그램 매출 비교 차트로 11개 지점 성과 순위 파악
  3. 목표 미달 지점은 빨간색으로 표시 → 클릭 시 해당 지점 상세 현황으로 이동
  4. 필요 데이터를 엑셀/PDF로 내보내기하여 경영진 보고에 활용

4.3 역할별 화면 구성

역할첫 화면 (메인)주요 사용 화면
매장 직원내 실적 요약 + 근태 현황근태 조회, 업무일지(AI 피드백), 내 실적, 목표 달성률, 내 평가 이력
매장 관리자지점 대시보드 (근태 + 실적 + 평가 요약)팀원 성과 관리, 근태/휴가 승인, 스케줄 관리, 상위자 평가 작성
본사 관리자본사 통합 대시보드 (전 지점 현황)지점별 비교, 직원 인사 관리, 평가 기간 관리, KPI/정책 설정

4.4 모바일 대응

매장 직원의 경우 스마트폰 사용 비율이 높을 것으로 예상되므로, 모바일 환경에서 아래 기능이 원활하게 동작하도록 최적화합니다.

  • 업무일지 작성: 모바일 키보드에 최적화된 입력 폼
  • 내 실적 확인: 세로 스크롤로 KPI 카드를 하나씩 확인하는 구조
  • 근태 현황 조회: 이번 달 출퇴근 기록, 근무시간 확인
  • 알림 확인: 앱처럼 알림 수신 가능

효과 영역기대 효과
