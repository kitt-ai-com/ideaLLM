> Enhans의 "업종×기능 AI 에이전트" 모델을 킷에이아이에 적용하면, 현재의 프로젝트 단가를 2~5배 높이고 월 구독 반복 매출(MRR)을 만들 수 있다. **1단계로 기존 강점 업종(물류, 의료, 교육)부터 에이전트를 패키징**하고, 2단계에서 커머스/브랜드 보호 영역으로 확장하는 전략이 현실적이다.

# 킷에이아이 AI 에이전트 사업 전략

## 전략 방향: 범용 자동화 → 업종 특화 AI 에이전트

### 현재 vs 목표

| 구분 | 현재 | 목표 |
|---|---|---|
| 포지셔닝 | 범용 AI 업무 자동화 | 업종 특화 AI 에이전트 솔루션 |
| 상품 구조 | 프로젝트별 맞춤 개발 | 에이전트 패키지 + 커스터마이징 |
| 수익 모델 | 프로젝트 단건 매출 | SaaS 구독형 MRR + 프로젝트 |
| 영업 방식 | "뭐든 자동화해드립니다" | "물류 출고 에이전트 도입하세요" |

## Enhans에서 벤치마킹할 핵심 전략

### 1. 에이전트 개별 상품화
- Enhans: 7종 에이전트를 각각 독립 상품으로 판매
- **킷에이아이 적용**: 기존 프로젝트 경험을 기반으로 에이전트 패키징
  - 물류 출고 자동화 에이전트
  - 병원 환자관리 에이전트
  - 교육기관 운영 에이전트
  - 브랜드 보호 에이전트 (신규)

### 2. 도입 사례 중심 마케팅
- Enhans: 업종×에이전트별로 구체적 ROI 수치 제시
- **킷에이아이 적용**: 기존 성과를 에이전트별 비즈니스 케이스로 재구성
  - "물류 출고 에이전트: 월 54시간 절감, 비용 35% 절감"
  - "병원 환자관리 에이전트: 재방문율 35% 향상"

### 3. Land & Expand 전략
- Enhans: 에이전트 1개 도입 → 성과 확인 → 추가 에이전트 도입
- **킷에이아이 적용**:
  - 1차: 핵심 에이전트 1개로 도입 장벽 낮춤
  - 2차: 성과 입증 후 추가 에이전트 제안
  - 3차: 플랫폼(kittOS?) 통합 제안

### 4. 플랫폼화
- Enhans: CommerceOS 위에 에이전트 동작
- **킷에이아이 적용**: 장기적으로 kittOS(가칭) 플랫폼 구축
  - n8n + AI 에이전트를 통합하는 자체 플랫폼
  - 에이전트 간 데이터 공유 및 워크플로우 연동

## 실행 로드맵

### Phase 1: 패키징 (1~2개월)
- 기존 고객 사례를 에이전트 상품으로 재패키징
- 에이전트별 랜딩페이지/비즈니스 케이스 제작
- 가격 체계 수립 (월 구독형)

| 에이전트 | 기반 사례 | 타겟 업종 |
|---|---|---|
| 출고 자동화 에이전트 | 물류/유통 사례 | 물류, 유통, 이커머스 |
| 환자관리 에이전트 | 병원/클리닉 사례 | 병원, 클리닉, 한의원 |
| 학원 운영 에이전트 | 교육/학원 사례 | 학원, 교육기관 |

### Phase 2: 확장 (3~6개월)
- 브랜드 보호 에이전트 개발 (Enhans 모델 참고)
- 커머스 영역 진출: 가격 모니터링, 리뷰 관리
- 업종별 비즈니스 케이스 축적

### Phase 3: 플랫폼화 (6~12개월)
- kittOS 통합 플랫폼 출시
- 에이전트 마켓플레이스
- 파트너/리셀러 프로그램

## 수익 모델 설계

### 현재
```
프로젝트 단건: 500만원~3,000만원 (일회성)
```

### 목표
```
에이전트 구독: 월 50만원~300만원/에이전트 (반복 매출)
+ 초기 세팅비: 200만원~500만원
+ 커스터마이징: 별도 견적
```

### 예상 MRR 시뮬레이션
| 시점 | 고객 수 | 평균 월 구독 | MRR |
|---|---|---|---|
| 3개월 후 | 5 | 100만원 | 500만원 |
| 6개월 후 | 15 | 150만원 | 2,250만원 |
| 12개월 후 | 40 | 200만원 | 8,000만원 |

## 마케팅 전략

### 웹사이트 리뉴얼
- Enhans처럼 에이전트별 개별 페이지 구성
- 업종×에이전트 비즈니스 케이스 강조
- ROI 수치를 전면에 배치

### 콘텐츠 마케팅
- 각 에이전트별 도입 사례 블로그
- "AI 에이전트로 물류 비용 35% 절감한 방법" 류의 구체적 타이틀
- 웨비나/교육 콘텐츠 (기존 Education 서비스 연계)

### 영업 전략
- 기존 고객에게 에이전트 구독 전환 제안 (업그레이드)
- 업종별 타겟 리드 발굴
- 무료 진단 → 에이전트 도입 제안 → 구독 전환
