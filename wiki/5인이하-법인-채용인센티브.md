# 5인 이하 법인 채용 인센티브 가이드

> 작성 기준: 2026-05-04 | 대상: 킷에이아이(현 3명, 채용 후 5명) 등 영세 AI/SW 법인

## "인원 기준" 정확히 구분

한국 노동법·고용지원금에는 **세 가지 인원 기준**이 섞여 있어 혼동이 큼.

| 기준 | 의미 | 핵심 영향 |
|---|---|---|
| **5인 미만** | 1~4명 | 근로기준법 일부 미적용 (연차·연장수당·해고예고 등) |
| **5인 이상** | 5~9명 | 근로기준법 전면 적용 + 일부 장려금 자격 시작 |
| **10인 미만** | 1~9명 | 두루누리 등 영세사업장 지원 대상 |
| **우선지원대상기업** | 업종별 다름 (서비스업 300명 이하) | 거의 모든 고용지원금의 기본 자격 |

> 킷에이아이 단계: **현재 3명 (5인 미만)** → 1명 채용 시 4명 (5인 미만) → 2명 채용 시 **5명 (5인 이상 진입!)**

## 인센티브 11종 (사업장 규모별 분류)

### A. 사업장 규모 무관 (5인 미만/이하 모두 가능) — 7개

#### ① 통합고용세액공제 ★★★★★

|---|---|

| 인원 조건 | **없음** (1인 사업장도 가능) |
| 킷에이아이 | 3→5명 시 **9,300만원 세액공제** |
| 신청 | 법인세 신고 시 (세무사 협업) |

## 채용 시퀀스 전략

### 전략 A: 단계적 채용 (자금 부담 분산)
```
3명 → [1명 청년 채용] → 4명 (우대업종 장려금)
     → 6개월 후 [1명 청년 채용] → 5명 (본 트랙 장려금)
```
- 두 채용 모두 청년일자리도약장려금 + 통합고용세액공제 청구 가능
- 6개월 시차로 사업장 적응 부담 분산
- **매출 4,000만원 단계 권장**

### 전략 B: 2명 동시 채용 (즉시 5인 진입)
- 빠른 캐파 확보, 자격 즉시 충족
- 자금 부담 큼 → **매출 6,000만원 단계 권장**

### 전략 C: 신중년 + 청년 조합
- 첫 1명 만 50세 이상 → 신중년 적합직무 장려금 연 960만원 + 일반 통합고용세액공제(770만원/년 × 3년)
- 두번째 청년 → 청년일자리도약장려금 추가
- 시니어 경험 + 정부지원 조합 효과

## 주의사항 (사전 확인 필수)

1. **임원 산정 방식**: "5인 이상" 기준에서 **등기임원은 제외될 수 있음** → 실질 근로자 4대보험 가입자만 카운트할 가능성. 사업장별 산정 방식 1350 직접 확인
2. **창업 후 5년 이내 우대업종**: 킷에이아이 해당 시 추가 가점/한도 상향 가능
3. **2026년 사업지침 확정 여부**: 매년 1월 갱신 — work24.go.kr 최신 지침 필수 확인
4. **중복수급 제한**: 청년일자리도약장려금 ↔ 고용촉진장려금 중복 불가, 단 세액공제와는 중복 가능
5. **TIPS·R&D 인건비**: TIPS/AI바우처 정부 매칭 인건비로 지급된 부분은 장려금 기준 임금에서 제외될 수 있음 → 이중 수혜 차단 규정 확인
