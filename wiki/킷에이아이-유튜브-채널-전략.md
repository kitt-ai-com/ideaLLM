> 킷에이아이 유튜브는 **"AI 실행 전문가가 실제로 만들어 보이는 채널"**로 포지셔닝한다. 한국어 중심, 주 1회 롱폼 + 숏폼 2~3개. **벤치마크 1순위는 조코딩AX파트너스·윤자동**(둘 다 유튜브→별도 B2B 법인 퍼널이 킷과 동일). 진입로는 경쟁자가 약한 **빈자리 — 이커머스 자동화(쇼핑몰·CS·정산·메타광고) + RAG 사내챗봇 실제 구축기 + 의사결정자용 AX ROI 콘텐츠**. 퍼널은 영상 → **무료 AI 자동화 진단** → 대행 계약으로 압축. **단 하나의 위험요소: "페이스리스"는 B2B 신뢰 목표와 충돌하므로 '진정성 페이스리스' 모델로 우회한다(§2).**

| 시장·언어 | **한국어 중심** | 국내 중소기업·이커머스 타깃, 전환율 최우선 |

## 1. 시장 진단 — 지금 한국 AI 유튜브의 그림

- **속도전 + 권위 서사가 성장 공식.** 윤자동은 토스 PO·18년 자동화 권위 + 최신 툴(Manus) 즉시 시연으로 **단일 영상 1,007만 뷰**를 터뜨려 1년 새 구독 2배(6.49만). 시민개발자 구씨는 **정확히 주 1회 + 최신 툴 즉시 다루기**로 1년 새 ~2배(5.55만).
- **검증된 한국형 B2B 퍼널 = "유튜브(무료) → 무료라이브/리드매그넷 → 강의·커뮤니티 → 별도 법인으로 B2B 구축·컨설팅".** (조코딩→조코딩AX파트너스, 윤자동→윤AI솔루션, 장피엠→AI Ground) — **킷에이아이는 이 구조의 본진(kitt.ai.kr 대행)을 이미 갖고 있다.**
- **B2B에서 조회수는 허영지표.** 한 사례는 구독 300~400명·영상당 수백 뷰로도 수십 건 미팅을 예약, 유튜브가 사업 최고 유입 채널이 됨. → **정확한 ICP 300뷰 > 무작위 3만 뷰.**

## 2. ⚠️ 가장 중요한 의사결정 — "페이스리스"의 함정과 해법

**리서치 3개 스트림이 일관되게 경고한다:**
- 윤자동·조코딩·구씨·장피엠 등 **성공한 한국 B2B 채널은 전부 talking-head + 실제 구축 시연.**
- *"AI 양산형 faceless 채널은 브랜드화 불가·신뢰 저하·툴 의존으로 한계가 명확하다"* (국내).
- *"2025.7 유튜브 가이드라인 개정 — 원본성 없는 페이스리스/AI보이스 콘텐츠 수익·추천 단속"* (해외).
- B2B 의사결정자의 **75%는 사양보다 '사고 리더십(사람의 전문성)'을 신뢰.**

> [!warning] 결론: "얼굴 안 나옴"은 가능하지만 "양산형 AI보이스 채널"은 목표(B2B 신뢰)를 스스로 깬다.

### 해법 — "진정성 페이스리스(Authentic Faceless)" 모델
얼굴은 안 내보내되, 신뢰는 **콘텐츠의 원본성**으로 만든다.

| 신뢰 요소 | 페이스리스로 구현하는 법 |
|---|---|
| 사람의 전문성 | **대표(유승범) 본인 실제 목소리 나레이션을 권장**(AI보이스보다 신뢰 압도적 ↑). AI보이스를 쓸 경우 → 단일 브랜드 보이스 고정 + 절대 기계적 listicle 금지 |
| 실전 증거 | 화면녹화의 **대상이 진짜여야 함** — `workKITT`·`sellKITT` 실제 대시보드, 실제 고객 프로젝트 빌드 과정, 실측 결과 숫자(87%·90%↓) |
| 차별성(반-양산형) | 남의 화면 캡처·뉴스 읽기 ❌ → **우리가 직접 만든 워크플로우/코드/결과**만. 유튜브 단속의 핵심은 "원본성 없음"이지 "얼굴 없음"이 아님 |
| 인격 | 손(핸즈온 시연), 음성 톤, 일관된 인트로/아웃트로, 케이스 스토리텔링 |

> [!tip] 권장 진화 경로
> **0~3개월**: 100% 페이스리스(화면녹화 + 본인 목소리). **4개월+**: 케이스 스터디·고객 인터뷰 영상에 한해 **대표 얼굴/실루엣 일부 노출**을 옵션으로(전환 영상에서만이라도). 신뢰가 곧 리드 전환율이다.

## 4. 포지셔닝 & 진입 전략 — 빈자리를 친다

> **채널 한 줄 정의:** "중소기업·이커머스의 반복업무를, AI로 진짜 없애 보이는 채널 — 킷에이아이"

n8n/ChatGPT **입문은 이미 레드오션**(패캠·인프런·노마드·러닝스푼즈 다 보유). 경쟁자가 약한 **4개 빈자리**를 공략한다:

| # | 빈자리 | 근거 | 킷 자산 매핑 |
|---|---|---|---|
| 1 | **이커머스 도메인 특화 자동화** (쇼핑몰 상세페이지·CS·정산·메타광고) | 윤자동·장피엠이 약한 영역 | `sellKITT`, 쇼핑몰/메타광고 자동화 실적 |
| 2 | **RAG 사내 지식챗봇 "실제 납품 구축기"** | B2B 수요 폭발 vs 유튜브 콘텐츠 희소 = 최대 빈자리 | RAG 87% 검색정확도 |
| 3 | **의사결정자(대표·팀장)용 AX ROI·도입 콘텐츠** | 기존 콘텐츠는 개인 부업/실무자 편중 | AI 진단·로드맵 컨설팅 |
| 4 | **AI 에이전트·바이브코딩 실전** (상승세 편승) | Manus·Claude Code 급상승 | 건강이, 견적 자동화 |

## 5. 콘텐츠 전략 — 3대 기둥 × Big 5

### 5-A. 콘텐츠 3기둥 (인지:전환 ≈ 2:1)
- **① Attract(검색·에버그린)**: ICP가 실제 검색하는 키워드. 복리로 누적되는 B2B 핵심 자산.
- **② Engage(추천·신뢰)**: 트렌드·아키텍처·프로세스 공개로 전문성 각인.
- **③ Convert(사회적 증거)**: 고객 케이스·결과 공개. 조회수 최저·전환율 최고.

### 5-B. "They Ask, You Answer" Big 5 (B2B 구매 80%는 영업 전에 끝난다)
가격 / 문제·실패사례 / 비교(vs) / 리뷰 / 베스트 — **이 5개를 솔직하게 답하는 콘텐츠가 한국 B2B에서 가장 강력**(뻔한 "AI 도입 5가지"보다 심층·구체가 압도).

### 5-C. 첫 20개 영상 기획안 (좁은 서브토픽 × Big5 × 3기둥)
> 원칙: 무작위 20개 ❌ → **좁은 주제군 안에서** 채워야 견인 속도 2배.

**Attract (1~10) — 검색 유입**
1. [How-to] n8n으로 쇼핑몰 CS 자동응대 만들기 (실시간 시연)
2. [비용] AI 업무자동화 외주, 진짜 얼마 드나 — 견적 기준 공개
3. [vs] n8n vs Make vs Zapier — 실무자가 정한 결론
4. [문제] AI 도입 실패하는 회사들의 공통점 5가지
5. [How-to] 사내 문서 검색봇(RAG) 30분 만에 붙이기
6. [베스트] 2026 중소기업 업무자동화 툴 TOP 7
7. [vs] 사내 개발 vs SaaS 구독 vs 외주 — 의사결정 프레임
8. [How-to] 엑셀·정산 반복업무 AI로 없애기
9. [문제] AI 도입 시 놓치기 쉬운 보안 체크리스트
10. [ROI] AI 자동화 ROI 계산법 (무료 템플릿 제공)

**Engage (11~16) — 신뢰 구축**
11. AI 에이전트가 "실제로 일하게" 만드는 아키텍처 해부
12. 우리가 고객사 프로젝트를 진행하는 전체 프로세스 공개
13. AX 로드맵: 어디서부터 시작해야 하나 (대표 관점)
14. 라이브 빌드: 구독자 요청 자동화 1개 완성
15. 견적서 자동화 시스템 만드는 과정 전부 (90%↓ 어떻게)
16. 자주 묻는 AI 도입 질문 10개 답변

**Convert (17~20) — 사회적 증거**
17. OO기업 업무 50% 자동화한 전체 과정 (케이스 스터디)
18. 고객 인터뷰 — "도입 전/후 무엇이 바뀌었나"
19. 무료 AI 진단 받으면 뭘 알 수 있나 (데모)
20. 직접 만들 사람 vs 맡길 사람 — 솔직 가이드

> 10개 발행 후 **CTR 6%+ 또는 AVD 55%+ 영상의 포맷을 찾아 다음 10개에 복제.**

### 5-D. 검증된 한국어 후킹 패턴
금액·성과형("월 OOO만원 아낀 자동화"), 호기심·도발형("이걸 몰랐을 테니까"), 권위·반전형("상위 1%는 ~", "충격, 이건 진짜"), 무코딩형("코딩 1도 몰라도"), 케이스형("고객사 OO 자동화 인터뷰"). **제목 핵심 키워드는 앞 40자 안 + 구체 숫자/시간.**

## 6. 포맷·제작 시스템

| 항목 | 권장 | 근거 |
|---|---|---|
| 롱폼 길이 | **8~15분** (교육 심화는 15분+), 6분↑엔 챕터 | 교육/How-To 유지율 1위(42.1%), AVD 50%+면 추천 3배 |
| 숏폼 | 주 2~3개, **티저형(클리프행어) + 롱폼 링크 연결** | 완결형보다 티저형이 3~5배 전환, 링크 시 구독전환 +40% |
| 첫 60초 | 사활 — 시청자 55%가 1분 내 이탈 | 첫 30초 재편집만으로 AVD 5~10%p↑ |
| 패키징 | **시간의 30%를 아이디어→제목→썸네일에**(영상 제작 전에) | CTR이 유튜브 중요도의 40%. Studio "Test&Compare" A/B 필수 |
| 썸네일 | 텍스트 2~4단어, 고대비색, 툴명 명시 | 신규 채널 검색 CTR 8~20% 가능 |
| 배치 제작 | **월 4개를 2시간 1세션에 몰아 녹화** | 분산 금지, 번아웃이 채널 포기 1위 원인 |

### 1 롱폼 → 멀티 리퍼포징 (콘텐츠 곱셈)
10분 롱폼 1개 → 숏폼 5~10개 + 블로그/SEO 1편 + 링크드인 5~10포스트 + 뉴스레터 1회 + 인용그래픽. (킷의 마케팅 자동화 역량으로 이 과정 자체를 반자동화 → "우리가 우리 도구로 한다"는 메타 증거)

## 7. 퍼널 — 영상에서 대행 계약까지 (안 팔리게 판다)

> 대원칙 (Hormozi): **"비밀(What)은 공짜로 다 주고, 실행(How/Who)을 팔아라."** 1%만 직접 하고 99%는 결국 맡긴다.

```
유튜브 롱폼 (가치 100% 제공)
  → 무료 리드매그넷으로 이메일/연락처 수집
  → 카카오 오픈채팅·뉴스레터 커뮤니티 (신뢰 누적)
  → "무료 30분 AI 자동화 진단" CTA   ← 가장 강력한 비살상형 B2B CTA
  → 진단 통화 → 제안 → 대행 계약 (kitt.ai.kr 본진)
```

- **킷 전용 리드매그넷**: ① 영상에서 만든 n8n/Make 워크플로우 JSON 무료 배포 ② "중소기업 AI 도입 로드맵 노션 템플릿" ③ "업무자동화 ROI 계산기 / 보안 체크리스트" ④ **"무료 AI 자동화 진단"**(핵심 CTA).
- **비살상형 CTA 규칙**: 본문은 100% 가치, 서비스 언급은 마지막 20~30초에 1회("직접 하기 어렵거나 시간 없으면 무료 진단 받아보세요"). **단일 CTA가 복수 CTA보다 클릭 371%↑.**
- **CTA 통일**: 음성 멘트 + 화면 텍스트 + end screen + 설명란 첫 2줄 + 고정 댓글이 모두 같은 다음 행동.
- **댓글→DM 자동화**: "댓글에 '진단' 쓰면 자료 DM" (ManyChat/n8n) → 알고리즘 부스트 + 리드 캡처 동시. **킷의 자동화 역량을 자가 시연하는 셈.**
- **B2B는 별도 브랜드/창구로 분리** (조코딩AX·윤AI솔루션 패턴) — 교육/콘텐츠 톤과 영업 톤을 분리.

## 8. 알고리즘 현실 & 신규 채널 0→1 (2025~2026)

- **2025.12 대변동**: 홈피드 롱폼 추천 슬롯 ~80% 축소, 그 자리를 Shorts가 차지. **Shorts↔롱폼 랭킹 모델 분리** → 숏폼 성과가 롱폼을 자동으로 안 올려준다. **연결은 의도적 설계 필요**(같은 주제·end screen·고정댓글).
- **"구독자 X명 필요"는 미신.** 구독 0인 채널 영상도 노출된다. 신규 영상은 소규모에 먼저 테스트 → 초기 신호(시청시간+CTR+만족도) 강하면 확장. **2026은 작은 채널을 더 빨리 테스트** + 마이크로 니치 클러스터링 → **좁은 니치 + 높은 만족도**가 B2B에 유리.
- **태그는 거의 무의미** → 제목·설명·자막(대사)에 키워드 집중. 잘 랭크된 1개 영상이 **월 50~200 리드를 무기한** 생성 가능.
- **첫 100~500 구독은 외부에서 직접 데려와라**(이메일·인스타·링크드인·기존 고객). 첫 10개 영상은 모든 댓글 1시간 내 답글.

## 9. 로드맵 & KPI

### 단계별 로드맵
| 시점 | 할 일 | 현실적 기대치 |
|---|---|---|
| **0~3개월** | 니치 확정, 어트리뷰션 셋업, 리드매그넷 1개, 첫 10개 영상(월 4개 배치), 패키징 훈련 | 월 20~50 구독. **조회수 말고 노출·CTR을 본다** |
| **3개월** | 분석 → 잘 된 포맷 복제, 무료진단 랜딩 가동 | **첫 예약 미팅 등장** |
| **6개월** | 에버그린 라이브러리 누적, 케이스 스터디 영상 추가 | 파이프라인 기여 가시화 |
| **6~12개월** | 주 1+ 유지, 리퍼포징 시스템화 | **1,000 구독 도달 가능**(영상 70~80개) |

### KPI — 선행(주간 최적화) vs 후행(비즈니스 증명)
- **선행**: 노출 CTR(검색 8~15%·전체 5~7%+), AVD(교육형 40~55%), 첫 60초 유지율(65%+), 댓글 질, 리드매그넷 전환율, 이메일/커뮤니티 순증.
- **후행**: **예약 미팅(무료진단) 수**, 유튜브 기인 파이프라인/매출, **리드 품질(ICP 적합도)**, CAC/LTV.
- **어트리뷰션(런칭 전 필수)**: 모든 링크에 UTM(source/medium/campaign) + **문의 폼에 "어떻게 알게 되셨나요?" 자유텍스트 필드.** 유튜브 리드는 같은 세션에 전환하지 않으므로 CRM이 놓치는 걸 자가보고로 잡는다.

## 10. 즉시 실행 체크리스트 (첫 90일)

- [ ] 채널 한 줄 정의 확정 + 경쟁 5채널(윤자동·구씨·장피엠·기묘한자동화·조코딩) 아웃라이어/댓글 분석으로 키워드 갭 발굴
- [ ] "진정성 페이스리스" 제작 가이드 확정 — 본인 목소리 vs AI보이스 결정(§2), 실제 화면 소스 목록화(workKITT·sellKITT·건강이)
- [ ] 어트리뷰션 셋업(UTM + "어떻게 알게 되셨나요" 필드) + 무료진단 랜딩페이지
- [ ] 리드매그넷 1개 제작(추천: "업무자동화 진단 체크리스트" 노션 + n8n 템플릿)
- [ ] 첫 10개 영상 = 빈자리(이커머스 자동화·RAG 구축기) × Big5 × Attract/Engage/Convert
- [ ] 제작 시스템: 월 4개 2시간 배치 녹화, 패키징(제목→썸네일)에 시간 30%
- [ ] 주 1 롱폼(8~15분) + 숏폼 2~3개(티저·롱폼 링크) — 첫 60초 사활
- [ ] 1 롱폼 → 숏폼/블로그/링크드인/뉴스레터 리퍼포징(킷 자동화 도구로 반자동)
- [ ] 90일 분석: CTR 6%+/AVD 55%+ 포맷 복제, 첫 미팅 추적
- [ ] **허영지표(조회수·구독) 아닌 리드 품질로 판단**
