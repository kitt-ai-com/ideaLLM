> [!takeaway] 핵심 테이크어웨이
> Enhans의 마케팅은 **"업종×에이전트×ROI 수치"** 공식으로 이루어진다. 킷에이아이도 기존 고객 성과 데이터를 이 공식에 맞춰 재구성하면, 웹사이트 리뉴얼 없이도 즉시 마케팅 효과를 높일 수 있다.

# 킷에이아이 에이전트 마케팅 전략

## Enhans 마케팅 구조 분석

### 페이지 구조 (벤치마킹 대상)
```
메인 → CommerceOS (플랫폼) → AI Agent (7종 개별 페이지)
                             → Impact Biz. Case (업종×에이전트 조합)
                             → Builder / Web Agent
```

### Enhans의 비즈니스 케이스 페이지 공식
```
1. Overview: 한 줄 요약 (누가 + 무엇을 + 어떤 성과)
2. Industry Challenges: 4가지 문제점 (탭 형식)
3. Solution: 솔루션 다이어그램
4. 상세 기능: 모니터링 → 분석 → 대응 3단계
5. Business Outcomes: 고객 증언 + ROI 수치 (97%, 4%, 6배)
6. CTA: Contact 버튼
```

> 이 공식은 B2B SaaS 마케팅의 정석. 킷에이아이도 동일 구조를 차용해야 한다.

## 킷에이아이 마케팅 액션 플랜

### 즉시 실행 (1~2주)

#### 1. 기존 성과를 Enhans 공식으로 재구성
현재 흩어져 있는 성과 데이터를 업종×에이전트 형태로 정리:

| 비즈니스 케이스 | 제목 예시 |
|---|---|
| 물류×출고 에이전트 | "물류 기업이 AI 출고 에이전트로 월 54시간을 절감한 방법" |
| 의료×환자관리 에이전트 | "클리닉이 AI 환자관리 에이전트로 재방문율을 35% 높인 방법" |
| 교육×학원운영 에이전트 | "학원이 AI 운영 에이전트로 업무 효율을 3배 높인 방법" |

#### 2. 웹사이트 구조 변경
```
현재: kitt.ai.kr
  - AI Development
  - Automation
  - Consulting
  - Education

목표: kitt.ai.kr
  - AI Agents (에이전트별 개별 페이지)
    - 출고 자동화 에이전트
    - 환자관리 에이전트
    - 학원운영 에이전트
    - 브랜드 보호 에이전트 (신규)
  - Impact Case (업종×에이전트 비즈니스 케이스)
    - 물류 × 출고 에이전트
    - 의료 × 환자관리 에이전트
    - 교육 × 학원운영 에이전트
  - Solutions (컨설팅 + 교육)
  - About
```

### 단기 실행 (1~2개월)

#### 3. 에이전트별 랜딩페이지 제작
각 에이전트마다 Enhans 공식을 따르는 전용 페이지:

**페이지 구조 (템플릿)**
```
[Hero] 한 줄 가치 제안 + CTA
[Problem] 이 업종의 4가지 핵심 문제
[Solution] 에이전트 기능 다이어그램
[How It Works] 3단계 프로세스
[Results] ROI 수치 + 고객 증언
[Pricing] 플랜 비교표
[CTA] 무료 상담 신청
```

#### 4. 콘텐츠 마케팅
- 블로그: "AI 에이전트로 [업종] [문제] 해결한 방법" 시리즈
- 케이스 스터디 PDF (영업 자료)
- 웨비나: "30분 만에 알아보는 AI 에이전트 도입법" (기존 Education 연계)

### 중기 실행 (3~6개월)

#### 5. 브랜드 보호 에이전트 출시 마케팅
- K-뷰티, K-푸드 브랜드 대상 타겟 마케팅
- "쿠팡/네이버에서 비공식 판매자 때문에 고민이신가요?" 메시지
- 무료 브랜드 진단 서비스 → 리드 확보 → 에이전트 도입 제안

#### 6. 파트너 마케팅
- 세무/법무 사무소와 제휴 (브랜드 보호 법적 대응 연계)
- 이커머스 대행사와 제휴 (판매자 관리 연계)
- 물류 3PL과 제휴 (출고 자동화 연계)

## 메시지 전략

### 현재 메시지
> "반복 업무는 AI에게, 당신은 성장에만 집중하세요"

### 제안 메시지 (에이전트 중심)
> **메인**: "당신의 업무에 전문 AI 에이전트를 배치하세요"
> **서브**: "물류, 의료, 교육, 커머스 — 업종별 맞춤 AI 에이전트가 24시간 일합니다"

### 에이전트별 카피
| 에이전트 | 헤드라인 |
|---|---|
| 출고 자동화 | "매일 3시간 걸리던 출고 정리, 15분으로 줄여드립니다" |
| 환자관리 | "환자가 다시 찾아오는 클리닉을 만들어드립니다" |
| 학원운영 | "원장님은 교육에만 집중하세요. 운영은 AI가 합니다" |
| 브랜드 보호 | "내 브랜드, 누가 어디서 얼마에 팔고 있는지 알고 계신가요?" |
