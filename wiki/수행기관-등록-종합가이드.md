## 1. 혁신바우처 수행기관 (mssmiv.com)

### 개요
- 중소기업 대상 **컨설팅·기술지원·마케팅** 3개 분야 10개 프로그램 바우처
- 주관: 중소벤처기업진흥공단 (중기부)

### 등록 요건
- 대한민국 사업자등록 보유 법인/개인사업자
- 해당 분야 서비스 제공 역량 보유
- **명시적 업력·매출·인력 기준 없음** (단, 심사에서 평가)
- 국세/지방세 체납 없을 것

### 서비스 분야 (AI/IT 관련)
| 분야 | 프로그램 | AI/IT 관련도 |
|---|---|---|
| 컨설팅 | 경영혁신, 탄소중립, 융복합, 중대재해예방 | 중 |
| 기술지원 | 기술진단, 시험인증, 디자인 | 중 |
| 마케팅 | 홍보, 판로개척 | **높음** |

### 등록 절차
1. mssmiv.com 회원가입
2. [공급기업 신청(일반)] 메뉴 → 온라인 신청
3. 사업자 공동인증서 + 대표자 공동인증서 필요
4. 서류 업로드 → 심사 → 선정 통보

### 필요 서류
- 사업자등록증명원
- 법인등기부등본
- 최근 3개년 표준재무제표증명 (⚠️ 신생법인은 없음 → 설립 후 첫 결산 재무제표)
- 납세증명서
- 사업신청서·사업계획서
- 수행실적 증빙

### 모집 시기
- 연 4~5차 수시 모집 (2024년 기준 1~4차 진행)
- **주로 제조업 중심**이므로 AI/IT 서비스 기업은 분야 선별 필요

### 정산 방식
- 후지급: 서비스 제공 → 결과보고서 → 검수 → 국비 지원금 지급
- 부가세는 수혜기업이 수행기관에 별도 납부

### 킷에이아이 평가
```
적합도: ★★★☆☆
장점: 업력 제한 없음, 수시 모집
단점: 제조업 중심, 3개년 재무제표 요구(신생법인 불리), 수행실적 요구
전략: 다음 차수 모집 시 "기술지원-융복합" 또는 "마케팅" 분야로 신청
```

## 2. 수출바우처 수행기관 (exportvoucher.com)

### 개요
- 수출기업에 바우처 지급 → 수행기관 서비스 구매
- 주관: KOTRA(산업부), 중소벤처기업진흥공단(중기부)
- 메뉴판: 15~17개 분야, 약 7,500개 서비스

### 등록 요건
- 국내 사업자등록 보유 법인/개인사업자
- 해당 분야 서비스 제공 능력 및 전문성
- 사업 실적 및 수행 경험 (있으면 유리)
- 전문 인력 보유
- **명시적 업력·매출 기준 없음**

### 서비스 분야 (AI/IT 관련)
| 분야 | AI/IT 관련도 | 킷에이아이 적합도 |
|---|---|---|
| **테크서비스** | 매우 높음 | ★★★★★ |
| 통번역 | 높음 (AI 번역) | ★★★★☆ |
| 홍보/광고 | 높음 (디지털 마케팅) | ★★★★☆ |
| 조사/일반 컨설팅 | 중 | ★★★☆☆ |
| 디자인개발 | 중 | ★★☆☆☆ |

**테크서비스 바우처**: AI, SaaS, 플랫폼 기업의 해외진출 전용. 업체당 최대 3,500만원, 국고 70%.

### 등록 절차
1. exportvoucher.com 회원가입
2. [수행기관 신청] → 서비스 등록
3. 서비스 요청 양식 등록
4. 제출서류 업로드
5. 심사 → 선정 (연 6회 심사)

### 필요 서류
- 서비스 수행 계획서 (서비스별)
- 사업실적 증명서
- 수행 인력 현황표 + 경력증명서
- 사업자등록증
- 특수관계기업 여부 확인서
- 정보제공 동의서, 부패방지 서약서

### 모집 시기
- **상시 모집** (연중 수시 접수)
- 선정 심사: 연 6회 (약 2개월 간격)

### 정산 방식
- 서비스 완료 → 참여기업 확인 → 정산 신청 (완료일 1개월 이내) → 익월 10일 입금
- 선금: 공급가액 1천만원 초과 + 계약 2개월 이상 시 최대 50%
- VAT는 참여기업이 수행기관에 별도 납부

### 킷에이아이 평가
```
적합도: ★★★★☆
장점: 상시 모집, 테크서비스 분야 정확히 해당, 업력 제한 없음
단점: 수출 연계성 입증 필요, 사업 실적 아직 없음
전략: "테크서비스" 분야로 LLM 기반 해외향 고객 대응 자동화 서비스 등록
     또는 "홍보/광고" 분야로 AI 기반 해외 디지털 마케팅 자동화 등록
```
