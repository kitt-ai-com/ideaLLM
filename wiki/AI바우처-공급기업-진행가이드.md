## 대안 루트 3가지

### 루트 1: 하반기 추가모집 노리기 (가능성 높음)

**근거:** 과거 AI바우처 사업은 매년 2~3차 추가모집을 진행했다.
- 2025년: 1차 → 2차 → 3차(최종) 모집 진행
- 2026년: 이미 1차, 2차 진행 → **하반기 3차 추가모집 가능성 높음**

**준비 일정:**
| 시기 | 작업 |
|---|---|
| 4~5월 | AI솔루션 상품화 완료, 데모 준비 |
| 6~7월 | 추가모집 공고 모니터링 시작 |
| 공고 즉시 | 신청서 + 솔루션 설명자료 제출 |

**모니터링 채널:**
- NIPA 사업공고: https://www.nipa.kr/home/bsnsAll/1/nttList?bbsNo=4&bsnsDtlsIemNo=580&tab=2
- 기업마당: https://www.bizinfo.go.kr
- 담당자 직접 문의: 043-931-5760 / ai-voucher@nipa.kr

### 루트 2: 수요기업 측으로 먼저 참여

킷에이아이의 **기존 고객사(물류, 병원, 학원)**가 수요기업으로 신청하고, 다른 POOL 등록 공급기업과 컨소시엄을 구성하는 방법. 킷에이아이는 부분 참여(컨설팅/PM)로 경험 축적.

→ 다만 이것보다는 루트 1이 더 직접적.

### 루트 3: 2027년 선제 준비 (확실한 루트)

2027년 1차 모집(예상: 2027년 1~2월)에 **완벽하게 준비된 상태**로 신청.

## 공급기업 POOL 등록에 필요한 것

### 1. 자격 요건

| 요건              | 킷에이아이 현황           | 충족 여부    |
| --------------- | ------------------ | -------- |
| 국내 AI 기업(법인)    | 주식회사 킷에이아이         | ✅        |
| 자체 개발 AI솔루션 보유  | **현재 없음 → 상품화 필요** | ⚠️ 준비 필요 |
| 상용화 실적 or 출시 계획 | 고객 사례 있음, 상품화는 안 됨 | ⚠️ 준비 필요 |
| 정부과제 참여제한 아닐 것  | 해당 없음 (추정)         | ✅        |
| 완전자본잠식 아닐 것     | 확인 필요              | ❓ 확인     |

### 2. 제출 서류 3종

#### ① AI바우처 공급기업 POOL 등록 신청서
- 기업 기본 정보
- AI솔루션 개요
- 기술 역량 / 인력 현황
- 수행 실적

#### ② 개인정보 수집 및 이용 동의서
- 표준 양식 (공고 시 제공)

#### ③ AI솔루션 세부 설명자료 ← **핵심**
- 솔루션명, 적용 분야
- 기술 구조 (아키텍처)
- AI 기술 요소 (어떤 AI를 쓰는지)
- 기능 설명 + 스크린샷/데모
- 도입 사례 / 실증 결과
- 차별화 포인트

### 3. 등록 평가 기준

```
등록신청 → 서류검토(적격성) → AI솔루션 전문가 검증 → 최종 등록
```

| 평가 항목 | 내용 | 킷에이아이 준비사항 |
|---|---|---|
| 적격성 | 법인 여부, 자본잠식, 참여제한 | 재무제표 확인 |
| AI솔루션 검증 | 실제 AI 기술 포함 여부, 완성도 | **솔루션 데모 준비** |

## 킷에이아이가 지금부터 준비할 것

### Phase 1: AI솔루션 상품화 (4~6월, 8주)

현재 프로젝트 단건 서비스를 **등록 가능한 AI솔루션 상품**으로 전환:

#### 솔루션 후보 3개

**솔루션 A: kitt 출고 자동화 에이전트**
```
적용 분야: 물류/유통
AI 기술: LLM(Claude/GPT) + n8n 자동화
기능: 출고 데이터 자동 수집 → AI 분류/오류검출 → 보고서 생성
실증: 월 54시간 절감, 오류율 0%
```

**솔루션 B: kitt 환자관리 에이전트**
```
적용 분야: 의료/클리닉
AI 기술: LLM 기반 환자 분석 + 자동 커뮤니케이션
기능: 환자 데이터 분석 → 재방문 예측 → 맞춤 리마인드 자동화
실증: 재방문율 35% 향상
```

**솔루션 C: kitt 브랜드 보호 에이전트** (신규)
```
적용 분야: 이커머스/브랜드
AI 기술: 웹 크롤링 + LLM 분석 + 자동 신고
기능: 비공식 판매자 탐지 → 가격 위반 분석 → 자동 대응
실증: MVP 개발 필요
```

> **추천:** 솔루션 A(출고 자동화)가 가장 빠르게 상품화 가능 — 이미 실증 완료

#### 상품화 작업 체크리스트
- [ ] 솔루션명 확정 (예: "kitt Logistics Agent")
- [ ] 기술 아키텍처 문서 작성
- [ ] AI 기술 요소 명세 (LLM 모델, 자동화 엔진, 데이터 파이프라인)
- [ ] 데모 환경 구축 (실제 동작 시연 가능)
- [ ] 관리자 대시보드 UI
- [ ] 도입 사례 문서화 (성과 수치 포함)
- [ ] 가격표 작성 (바우처 규격에 맞춤)

### Phase 2: 실증 사례 강화 (4~7월)

POOL 등록 시 **"AI솔루션 전문가 검증"**을 통과하려면 실증이 핵심:

- [ ] 기존 고객에게 에이전트 형태로 서비스 전환 제안
- [ ] 성과 데이터 공식 수집 (Before/After 비교)
- [ ] 고객 추천서/증언 확보
- [ ] **최소 2건 이상의 도입 사례** 문서화

### Phase 3: 신청서 사전 작성 (7~8월)

추가모집 공고가 나기 전에 미리 준비:

- [ ] 기업 기본 정보 최신화 (사업자등록증, 재무제표 등)
- [ ] AI솔루션 세부 설명자료 완성 (10~15페이지)
- [ ] 데모 영상 촬영 (3~5분)
- [ ] 등록 시스템(pms.ai-voucher.or.kr) 계정 생성 및 사전 입력

### Phase 4: 추가모집 즉시 신청 (공고 시)

- 공고 확인 즉시 서류 제출 (이미 준비 완료 상태)
- POOL 등록 후 → 수요기업 발굴 → 컨소시엄 구성 → 과제 신청

## 과제 수행 시 알아야 할 것

### 과제 선정 평가 기준 (공급기업 역할)

| 평가 항목 | 비중 | 킷에이아이 대응 |
|---|---|---|
| **솔루션 우수성** | 높음 | AI 기술 차별성, 기능 완성도 |
| **참여인력 전문성** | 높음 | AI/자동화 전문 인력 이력서 |
| **후속지원 계획** | 중간 | 유지보수, 업데이트, 교육 계획 |
| **수행 실적** | 중간 | 기존 도입 사례 수/질 |

### 과제 수행 흐름
```
POOL 등록 → 수요기업과 매칭 → 컨소시엄 구성 → 과제 신청서 작성
→ 선정 평가 → 협약 체결 → 사업 수행 (7개월) → 정산/완료
```

### 자금 흐름
```
정부 → 수요기업에 바우처(최대 2억) 지급
수요기업 → 바우처로 공급기업(킷에이아이) 솔루션 구매
+ 수요기업 민간매칭(현금/현물) 추가
```

## 수요기업 발굴 전략

POOL 등록 후 가장 중요한 것은 **수요기업 찾기**:

### 타겟 수요기업

| 업종 | 솔루션 | 접근 방법 |
|---|---|---|
| 물류/유통 중소기업 | 출고 자동화 에이전트 | 기존 고객 네트워크, 물류 협회 |
| 병원/클리닉 | 환자관리 에이전트 | 기존 고객, 의료 컨설팅사 |
| 이커머스 브랜드 | 브랜드 보호 에이전트 | K-뷰티, K-푸드 브랜드 직접 영업 |
| 학원/교육기관 | 학원운영 에이전트 | 기존 고객, 교육 플랫폼 |
