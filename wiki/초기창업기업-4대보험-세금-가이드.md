## 매출 없을 때 세금 처리

| 항목 | 내용 |
|---|---|
| **부가세** | 매입세액(서버·AI API·소프트웨어) 환급 가능 — 오히려 받는 구조 |
| **법인세** | 매출 없으면 결손 → 납부 없음. 결손금 10년 이월 가능 (향후 절세 자산) |
| **원천세** | 급여 지급 시에만 발생 → 미지급 시 없음 |
| **대표 급여** | 이사회 결의 전까지 미설정 가능 → 소득세 없음 |

---

## 정부과제 연계 시 최적 전략

```
지금 (과제 신청 중)
  → 4대보험 가입 불필요
  → 부가세 환급용 세금계산서만 챙기기

과제 선정 확인 후 (D-30)
  → 대표·CIO 건강보험·국민연금 취득신고
  → 이사회 결의로 급여 최소금액 설정

과제 착수 (7월)
  → 신규 개발자 채용 → 4대보험 전체 가입
  → 첫 급여 지급 → 원천세 신고 (다음달 10일)

과제 수행 중 (7~12월)
  → 정부 사업비로 급여 + 4대보험 전액 충당
  → 자부담 없이 인건비 처리 가능
```
