> [!takeaway] 핵심 테이크어웨이
> Enhans는 "범용 AI 자동화"가 아닌 **커머스 특화 AI 에이전트 7종**을 개별 상품화하여 높은 단가와 반복 매출을 만들고 있다. 킷에이아이도 현재의 범용 자동화 서비스를 **업종×기능별 AI 에이전트로 패키징**하면 단가를 높이고 경쟁력을 확보할 수 있다.

# Enhans 비즈니스 모델 분석

## 개요
Enhans는 AI 기반 상거래(커머스) 자동화 솔루션 회사로, **CommerceOS**라는 운영 체제 위에 7종의 전문 AI 에이전트를 제공한다.

## 제품 구조

### CommerceOS (플랫폼)
모든 에이전트가 동작하는 통합 운영 체제. 에이전트 간 데이터 공유와 워크플로우 연동이 가능하다.

### AI Agent 라인업 (7종)
| Agent | 기능 | 해결하는 문제 |
|---|---|---|
| Price Agent | 가격 모니터링/최적화 | 경쟁사 대비 가격 경쟁력 |
| Promotion Agent | 프로모션 자동화 | 프로모션 기획/실행 비효율 |
| Social Media Agent | SNS 관리 | 다채널 SNS 운영 부담 |
| QA Agent | 품질 관리 | 제품 품질 이슈 대응 |
| **Brand Protection Agent** | 브랜드 보호 | 비공식 판매자/가격 유출 |
| Review Agent | 리뷰 관리 | 리뷰 모니터링/대응 |
| AD Agent | 광고 관리 | 광고 성과 최적화 |

### Builder
사용자가 직접 워크플로우를 구성할 수 있는 도구

### Web Agent (ACT-1, ACT-2)
웹 환경에서 자동으로 작업을 수행하는 에이전트

## 비즈니스 전략 핵심

### 1. 버티컬 특화
- "AI 자동화" 범용이 아닌 **커머스(상거래)**에 집중
- 이커머스, F&B, 패션 등 커머스 세부 업종별 사례 제시

### 2. 에이전트별 개별 상품화
- 하나의 플랫폼에서 7개 에이전트를 독립 상품으로 판매
- 고객은 필요한 에이전트만 선택 도입 → 점차 확장 (Land & Expand)

### 3. 도입 사례 중심 마케팅
- Impact Biz. Case 페이지에서 업종×에이전트 조합별 성과를 구체적 수치로 제시
- 97% 블랙셀러 차단, 4% 이익률 개선 등 ROI가 명확

### 4. 글로벌 스케일
- 50개국, 1,000+ 플랫폼 모니터링으로 글로벌 기업 타겟
- 다국어 지원 (KOR/ENG)

## 수익 모델 (추정)
- SaaS 구독형 (월/연 과금)
- 모니터링 범위(플랫폼 수, SKU 수)에 따른 티어별 가격
- 에이전트 추가 도입 시 번들 할인
