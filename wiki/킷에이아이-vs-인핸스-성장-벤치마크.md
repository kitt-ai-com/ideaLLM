> [!takeaway] 핵심 테이크어웨이
> 인핸스의 5년 성장 공식을 킷에이아이에 적용하면: **①업종 특화(물류/의료/교육) ②기술력 증명(자동화 성과 수치화) ③레퍼런스 고객 1~2개 확보 ④정부 지원사업/액셀러레이터 활용**이 핵심이다. 인핸스가 Pre-A에서 시작해 5년 만에 290억 투자를 유치한 것처럼, 킷에이아이도 단계별 성장이 가능하다.

# 킷에이아이 vs 인핸스 — 성장 벤치마크

## 현재 위치 비교

| 항목 | 인핸스 (현재) | 킷에이아이 (현재) | 격차 |
|---|---|---|---|
| 설립 | 2021년 (5년차) | - | - |
| 임직원 | 82명 | 소규모 | 큰 격차 |
| 누적 투자 | 290억원 | - | - |
| 매출 | 88억원 (2025) | 프로젝트 단건 | 큰 격차 |
| 제품 | CommerceOS + Agent 7종 | 범용 자동화 서비스 | 상품화 격차 |
| 고객 | 삼성전자, P&G 등 20개+ | 물류, 의료, 교육 소수 | 레퍼런스 격차 |
| 기술 | LAM, 특허 23건 | GPT/Claude 활용, n8n | 독자 기술 격차 |
| 글로벌 | 50개국, SF 사무소 | 국내만 | - |

## 인핸스의 성장 단계를 킷에이아이에 매핑

### 인핸스 Year 1~2 (2021~2022) → 킷에이아이 NOW

**인핸스가 한 것:**
- 바이럴픽으로 시작, 커머스 데이터 자동화에 집중
- 핵심 기술 개발 (상품 정보 크롤링, AI 분석)
- Pre-A 25억 유치

**킷에이아이가 해야 할 것:**

#### Step 1: 버티컬 선택 & 집중 (즉시)
인핸스가 "커머스"를 선택한 것처럼, 킷에이아이도 **하나의 버티컬**을 선택해야 한다.

| 후보 버티컬 | 기존 사례 | 시장 규모 | 추천도 |
|---|---|---|---|
| 물류/유통 자동화 | 출고 정리 자동화 | 대형 | ★★★★ |
| 의료/클리닉 자동화 | 환자관리 자동화 | 중형 | ★★★ |
| 교육/학원 자동화 | 학원 운영 자동화 | 중형 | ★★★ |
| **커머스/브랜드 보호** | 신규 | **초대형** | ★★★★★ |

> 추천: 기존 물류/유통 사례를 기반으로 **커머스 버티컬**로 확장. 인핸스와 같은 시장이지만 한국 중소 브랜드 타겟으로 차별화.

#### Step 2: 대표 에이전트 1개 완성 (1~3개월)
- 인핸스도 처음에는 "상품 정보 자동화" 하나로 시작
- 킷에이아이는 **출고 자동화 에이전트** 또는 **브랜드 보호 에이전트** 하나를 완성
- SaaS 구독 모델로 전환

#### Step 3: 성과 수치화 (3~6개월)
- 인핸스: "매출 10배 증가"로 시리즈A 유치
- 킷에이아이: 기존 고객 성과를 구체적 수치로 정리
  - "월 54시간 절감" → "연간 인건비 1,500만원 절감"
  - "재방문율 35% 향상" → "연간 매출 X천만원 증가"

### 인핸스 Year 3 (2023) → 킷에이아이 Year 1~2

**인핸스가 한 것:**
- 사명 변경 (바이럴픽 → 인핸스)
- 시리즈A 75억 유치 (매출 10배 성장이 근거)
- 삼성전자 고객 확보
- 특허 출원/등록

**킷에이아이가 해야 할 것:**

#### Step 4: 레퍼런스 고객 확보 (6~12개월)
- 인핸스의 "삼성전자" 같은 **누구나 아는 고객 1~2개**
- 물류: CJ대한통운, 쿠팡 풀필먼트 파트너 등
- 커머스: 아모레퍼시픽, CJ제일제당 등 K-브랜드

#### Step 5: 투자 유치 준비 (12~18개월)
- 정부 지원사업 참여 (팁스, 창업성장기술개발 등)
- 액셀러레이터 프로그램 (Plug and Play, 스파크랩 등)
- Pre-A 목표: 5~10억원

### 인핸스 Year 4~5 (2024~2025) → 킷에이아이 Year 2~4

**인핸스가 한 것:**
- 에이전트 7종 완성 → CommerceOS 플랫폼화
- 시리즈B 170억 유치
- 팔란티어 펠로우십, 국무총리상
- 매출 88억원

**킷에이아이 장기 목표:**
- 에이전트 3~5종 완성 → kittOS 플랫폼
- 시리즈A 투자 유치
- 매출 20~50억원 목표

## 킷에이아이 성장 로드맵

```
NOW ──────────────────────────────────────────→

[Phase 0: 지금] (0~3개월)
├─ 버티컬 선택 (커머스/물류)
├─ 대표 에이전트 1개 MVP
├─ 기존 고객 성과 수치화
└─ 웹사이트 리뉴얼 (에이전트 중심)

[Phase 1: 제품-시장 핏] (3~12개월)
├─ 대표 에이전트 정식 출시
├─ SaaS 구독 모델 전환
├─ 레퍼런스 고객 3~5개 확보
├─ 월 MRR 500만원+ 달성
└─ 정부 지원사업/액셀러레이터 참여

[Phase 2: 성장] (12~24개월)
├─ 에이전트 2~3종 추가
├─ Pre-A 투자 유치 (5~10억원)
├─ 팀 확장 (10~15명)
├─ 연 매출 10억원+ 달성
└─ 특허 출원 시작

[Phase 3: 스케일업] (24~48개월)
├─ kittOS 플랫폼 출시
├─ 시리즈A 투자 유치 (30~50억원)
├─ 팀 30~50명
├─ 연 매출 50억원+ 목표
└─ 해외 진출 준비
```

## 인핸스에서 배울 핵심 전략

### 1. "사이드 프로젝트에서 시작" 마인드셋
- 인핸스도 주말 사이드 프로젝트로 1.5년 실험 후 법인 설립
- 킷에이아이도 이미 고객 사례가 있으므로 실험 단계는 통과

### 2. 투자자 신뢰 = 재투자
- 인핸스의 초기 투자자 전원이 후속 라운드에 참여
- → 작은 금액이라도 첫 투자를 잘 활용해 성과를 보여주면 후속 투자 연결

### 3. "이름값"을 만드는 전략
- 인핸스: 삼성전자 → 팔란티어 → 국무총리상 → 이세돌 협업
- 킷에이아이: 유명 고객사 확보 → 정부 수상 → 업계 행사 발표

### 4. 기술 → 브랜딩 → 투자 선순환
```
독자 기술 개발 → 리더보드/특허로 증명 → 대형 고객 확보
    ↑                                         ↓
투자금으로 팀 확장 ← 투자 유치 ← 매출 성장
```

## 즉시 실행 가능한 3가지

1. **기존 고객 성과를 "에이전트 도입 사례"로 리패키징** — 1주일이면 가능
2. **대표 에이전트 1개 MVP 개발** — 기존 n8n + Claude 기술로 4~6주
3. **정부 지원사업 신청** — TIPS, 창업성장기술개발, AI 바우처 등 검토
