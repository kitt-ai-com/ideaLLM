# AI오픈데이터챌린지 (제2회 OpenData X AI 챌린지)

## 개요
제2회 OpenData X AI 챌린지. 공공데이터로 공공기관 과제를 AI로 해결. 킷에이아이는 **조흥(정책자금 융자신청서 자동생성, 중진공)** 과제 타겟. kitt 기존 수치 기반, 할루시네이션 3중 방어, PoC 정량목표 6개.

## 핵심
- **신청처(2중 접수)**: ① **K-Startup**(k-startup.go.kr) — 사업계획서 제출 / ② **나라장터**(g2b.go.kr) — 우승 시 계약용 입찰 등록 (계약연계형 필수)
- **마감**: 기본 6/5 → 연장됨(공식 재확인 필요). 문의 1357 / 창업진흥원 044-410-1991
- 신청자격: **창업 10년 이내** → 킷에이아이 충족 ✅
- 구조: 계약연계형 3과제(조흥=중진공·소흥=소진공·창흥=창진원) + 일반형 3과제
- 지원: 서류통과 과제별 5개사(총 30개사) PoC 2,000만원 + GPU 인프라 / 10주 실증 / 최종 과제별 1개사(총 6개사) 각 **1억원** + 후속계약 ~1억 + 창업패키지·초격차·R&D 우대
- 사업계획서 v4.0 12섹션 완성 → 최종 점검 후 제출만 남음
- 평가: 사용자 체험평가(중진공 직원) + 발표평가 병행
