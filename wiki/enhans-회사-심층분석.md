> [!takeaway] 핵심 테이크어웨이
> 인핸스는 5년 만에 "주말 사이드 프로젝트 → 누적 290억 투자 → 매출 88억 → 팔란티어 펠로우십 → 국무총리상"까지 도달했다. 핵심 성공 공식은 **①버티컬 특화(커머스) ②기술력 증명(리더보드 3위) ③글로벌 레퍼런스(삼성, P&G) ④초기 투자자 전원 재투자**다. 킷에이아이도 이 4가지 축을 만들어야 한다.

# Enhans(인핸스) 회사 심층 분석

## 한눈에 보기

| 항목 | 내용 |
|---|---|
| 설립 | 2021년 2월 (구 바이럴픽) |
| 대표 | 이승현 (KAIST 석사, 삼성전자 Bixby Vision) |
| 임직원 | 82명 (80% AI/SW 전문가) |
| 누적 투자 | **290억원** (Series B) |
| 2025년 매출 | **88억원** |
| 고객사 | 삼성전자, P&G 등 20개+ 엔터프라이즈 |
| 핵심 제품 | CommerceOS + AI Agent 7종 |
| 비전 | "커머스 분야의 팔란티어", 유니콘 목표 |

## 창업 스토리

### 대표 배경
- **이승현**: KAIST 컴퓨터공학 석사
- 삼성전자에서 **Bixby Vision** 담당
- 아마존과의 협업에서 커머스 AI의 잠재력 발견

### 창업 계기
1. 삼성전자 재직 중 아마존 협업 → "특정 기능 추가가 즉각적 거래 전환율 상승으로 이어짐" 체감
2. **"데이터와 실행이 연결될 때 비즈니스 효과가 극대화된다"** 는 핵심 통찰
3. 온라인 판매 업무 직접 경험 → 자동화 필요성 절감
4. 주말마다 개발자/엔지니어들과 사이드 프로젝트
5. **1년 반의 실험** 끝에 2021년 법인 설립 (바이럴픽)

> 킷에이아이 시사점: "실무 경험에서 나온 문제 의식 → 기술로 해결"이라는 창업 스토리는 투자자/고객 모두에게 강력한 설득력을 가진다.

## 투자 성장 타임라인

```
2021.02  법인 설립 (바이럴픽)
   │
2021.12  Pre-A   25억원  ─── 현대기술투자, L&S벤처캐피탈
   │
2023.09  Series A  75~90억원 ─── 기존 투자자 전원 + 타임폴리오, AOA캐피탈
   │     사명 변경: 바이럴픽 → 인핸스
   │     매출 전년 대비 10배 성장
   │     삼성전자 고객 확보
   │
2025.01  Series B  170억원 ─── L&S, KDB, 산업은행, KT인베스트먼트
   │     누적 290억원
   │
2025.05  팔란티어 스타트업 펠로우십 선정 (한국 유일)
   │     국무총리상 수상
   │     매출 88억원
   │
2026.01  NRF 2026 참가, 샌프란시스코 사무소 개설
   │     Microsoft AI Startup Alliance 체결
   │
2026.03  이세돌 9단 AI 협업 시연
         82명, 특허 23건, 고객사 20개+
```

### 투자 핵심 포인트
1. **초기 투자자 전원 재투자**: Pre-A → Series A → Series B 모두 기존 투자자 참여 → 강한 신뢰 시그널
2. **매출 성장률이 투자를 견인**: 시리즈A 당시 전년 대비 매출 10배 증가
3. **라운드 간격**: Pre-A→A (약 2년), A→B (약 1.5년) → 적절한 성장 속도
4. **투자자 다양화**: VC → CVC → 은행/금융 → 글로벌(팔란티어)

## 제품 진화 과정

```
[Phase 1] 바이럴픽 시절 (2021~2023)
  → AI 기반 상품 정보 관리/자동화
  → "AI가 잘 팔릴 상품을 찾아주는" 솔루션

[Phase 2] 인핸스 전환 (2023~2024)
  → 개별 AI 에이전트 개발 (Price, Brand Protection 등)
  → 삼성전자, P&G 등 대형 고객 확보

[Phase 3] CommerceOS 출시 (2025~)
  → 에이전트 7종을 통합하는 플랫폼(OS) 출시
  → 50개 기능, 1,000개 엔드포인트
  → LAM(Large Action Model) 기반으로 기술 차별화

[Phase 4] 글로벌 확장 (2026~)
  → 팔란티어 펠로우십, NRF 참가
  → 샌프란시스코 사무소, 50개국 진출 목표
  → "커머스의 팔란티어" 포지셔닝
```

## 핵심 성공 요인 분석

### 1. 버티컬 특화 전략
- "범용 AI"가 아닌 **커머스 한 분야**에 올인
- 커머스의 복잡한 문제(가격, 재고, 브랜드 보호, 리뷰)를 에이전트별로 해결
- → 깊은 도메인 전문성 → 높은 진입 장벽

### 2. 기술력 객관적 증명
- Act-1 에이전트: **Online-Mind2Web 리더보드 세계 3위** (구글 Gemini, OpenAI Operator 다음)
- 특허 23건 (미국/국내)
- LAM(Large Action Model) 독자 기술
- → 투자자와 고객 모두에게 기술력 신뢰

### 3. 레퍼런스 고객 확보
- **삼성전자, P&G** = 누구나 아는 글로벌 기업
- 고객사 사례: 연간 매출 2,000만 달러 증가
- → 다른 기업의 도입 결정을 가속화

### 4. 전략적 파트너십
- 팔란티어 스타트업 펠로우십 (글로벌 25개 중 한국 유일)
- Microsoft AI Startup Alliance
- → 글로벌 신뢰도 + 기술 연동

### 5. 팀 구성
- 80% AI/SW 전문가 (서울대, KAIST, 카카오, 쿠팡 출신)
- → 기술 중심 조직 = 빠른 제품 개발

## 현재 위치 & 향후 방향

### 현재 (2026년 기준)
- 매출 88억원, 임직원 82명, 누적 투자 290억원
- 국내 20개+ 엔터프라이즈 고객
- 샌프란시스코 사무소로 북미 진출 시작

### 향후 계획
- **단기**: 50개국, 500개+ 기업 대상 서비스 확대
- **중기**: 커머스 분야 유니콘 달성
- **장기**: 커머스 → 금융, 방위산업, 서비스 등 버티컬 에이전트 확장
