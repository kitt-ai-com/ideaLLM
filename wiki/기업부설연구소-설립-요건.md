> 기업부설연구소(또는 연구개발전담부서)는 **IITP AX혁신기업(최대 6.5억), ICT R&D, R&D 바우처의 선결조건**이다. 킷에이아이 같은 이공계 대표의 AI/SW 소기업은 자격요건이 까다롭지 않다(★☆☆☆☆) — **대표 1인만으로 연구개발전담부서를 오늘 바로 신고 가능**하다. 단 ①이공계 학사/기사 자격 ②유급+국민연금·건강보험 사업장가입(무보수 대표는 결격) ③독립 연구공간(칸막이+표지판) 3요건이 필수다. 부설연구소(2명)는 자격 임원 1명을 더 등재하면 창업 3년 이내 특례로 전환 가능하다.

# 연구소 설립 — 기업부설연구소 / 연구개발전담부서

## 개요

- **신고·인정 기관**: KOITA(한국산업기술진흥협회), 온라인 신고 (rnd.koita.or.kr)
- **종류**: ① 기업부설연구소(요건 높음) ② 연구개발전담부서(요건 낮음, 1명부터 가능)
- **핵심 요건**: ① 인적 요건(연구전담요원) ② 물적 요건(독립 연구공간 + 기자재)
- **위치**: R&D 정부지원사업의 진입 자격 → [[2026-전체-정부지원사업-총정리]] ICT R&D 선결조건

## 1. 인적 요건 (연구전담요원 수)

| 구분 | 필요 인원 |
|---|---|
| **연구개발전담부서** | 규모 무관 **1명 이상** ← 소기업·1인 기업 시작점 |
| 소기업 부설연구소 | 3명 이상 (**창업 3년 이내 소기업은 2명**) |
| 중기업 | 5명 이상 |
| 벤처기업·연구원창업기업 | 2명 |
| 중견·대기업 | 7~10명 이상 |

**연구전담요원 자격** (아래 중 하나):
- 자연계(자연·공학·의학) 분야 **학사 이상** — 컴공·전자·수학·통계 등 SW 분야 폭넓게 인정
- 국가기술자격 **기사 이상** (예: 정보처리기사)
- 전문학사 + 연구경력 2년 / 고졸 + 경력 4년 (중소기업 한정 완화)

### 대표이사도 연구전담요원이 될 수 있는가 → 가능 (조건부)

| 조건 | 내용 |
|---|---|
| 자격 | 일반 전담요원과 동일(이공계 학사/기사 등) |
| **유급 + 4대보험** | 사업장 국민연금·건강보험 직장가입자 + 보수 지급 |
| 상근 | 상시 출근, 타사 겸직 대표 불가 |
| 겸직 | **중소기업은 경영+연구 겸직 인정**(대기업은 엄격) |

### 대표 1인만으로 가능한가 → 전담부서는 가능, 부설연구소는 불가

| 형태 | 최소 인원 | 대표 1인 단독 |
|---|---|---|
| **연구개발전담부서** | 1명 | 🟢 가능 |
| 기업부설연구소(창업 3년 내 특례) | 2명 | 🔴 불가 (1명 부족) |

> ⚠️ 1인 대표가 유일 전담요원이면 KOITA가 "경영+연구 동시 전담 가능성"을 따져 보완요청할 수 있다. 신고서에 **연구과제·실적·연구공간**을 구체적으로 적어 실제 연구활동을 입증하면 통과율↑.

### 4대보험 가입 기준 (대표 핵심 함정)

연구전담요원 인정의 핵심은 "그 사업장에서 **유급 상근**인가"다. 대표이사는 4대보험 중 일부가 원래 가입 대상이 아니다.

| 보험 | 대표이사(사업주) | 인정 기준 |
|---|---|---|
| 국민연금 | 사업장가입자 가입 ✅ | **필수 확인** |
| 건강보험 | 직장가입자 가입 ✅ | **필수 확인** |
| 고용보험 | 사업주라 제외 ❌ | 미가입 정상, 결격 아님 |
| 산재보험 | 사업주라 제외 ❌ | 미가입 정상, 결격 아님 |

- **결격 사례**: 무보수 대표 / 지역가입자(사업장 아님) / 본사 무보수+타사 직장가입
- **확인·발급**: 4대사회보험 정보연계센터(www.4insure.or.kr) → 사업장가입자 자격득실확인서

## 2. 물적 요건 (연구공간 + 기자재)

- **독립된 연구공간**: 사방이 벽·칸막이로 막히고 **별도 출입문**이 있는 고정 공간, 다른 부서와 구분
- 면적 **50㎡ 이하**면 칸막이·**"연구개발전담부서" 표지판**으로 구분 인정 (소기업 사무실 흔한 방식)
- **연구기자재**: SW 기업은 개발용 PC·서버·SW 라이선스 등으로 충족 가능
- ⚠️ **탈락·보완요청 최다 사유가 공간 독립성** — 탁 트인 사무실 한 켠은 불인정

## 3. 신고 절차 (KOITA, 사전 신고제, 총 2~3주)

```
[준비 1~2일] → [온라인 신고 반나절] → [검토·발급 7~14일] → [사후관리]
```

| 단계 | 내용 |
|---|---|
| **1. 사전 준비** | 전담요원 졸업증명서/자격증 + 4대보험 가입증명 + 칸막이·표지판 연구공간 사진 + 기자재 목록 |
| **2. 회원가입** | KOITA 연구소인정시스템(rnd.koita.or.kr) 기업회원 가입 (사업자등록증) |
| **3. 신고서 작성** | 연구소/전담부서 개요 + **연구과제명·연구내용** + 조직도 입력 |
| **4. 증빙 첨부** | ①졸업·자격증명서 ②4대보험 가입증명 ③조직도 ④연구공간 도면·사진 ⑤사업자등록증 ⑥기자재 목록 |
| **5. KOITA 검토** | 7~14일, 보완요청 시 보완 후 재제출 |
| **6. 인정서 발급** | "기업부설연구소/연구개발전담부서 인정서" → R&D 사업 제출용 |
| **7. 사후관리** | 전담요원 변동·이전 시 변경신고, 매년 실태조사 응답 |

### 자격요건 까다로움 평가

| 요건 | 까다로움 | 비고 |
|---|---|---|
| 인적(자격) | 🟢 낮음 | 이공계 학사면 거의 자동 충족 |
| 물적(공간) | 🟡 중간 | 보완요청 최다 사유, 독립성 확보 필요 |
| 서류 | 🟢 낮음 | 발급 쉬운 서류뿐 |
| 심사 | 🟢 낮음 | 신고제, 7~14일 |

## 4. 주식회사 킷에이아이 적용 판정

현 조직: 임원 2 + 사원 1, AI/SW 기업, **자격 인력 1명, 창업 3년 이내** (확인됨, 2026-05-22)

| 형태 | 요건 | 킷에이아이 | 판정 |
|---|---|---|---|
| **연구개발전담부서** | 전담요원 1명 + 독립공간 | 자격 인력 1명(대표 포함 가능) | 🟢 **즉시 신고 가능** |
| **기업부설연구소** (창업 3년 내 특례) | 전담요원 2명 + 독립공간 | 1명 → 1명 부족 | 🟡 자격 임원 1명 추가 등재/채용 시 가능 |

### 권장 실행 순서

1. **지금: 연구개발전담부서부터 신고** 🟢 — 대표(이공계·유급·상근) 1인 + 칸막이 연구공간이면 충족. R&D 자격의 핵심 확보
2. **승격: 부설연구소** 🟡 — 자격 임원 1명을 더 등재(또는 자연계 학사/기사 개발자 채용)하면 **창업 3년 이내 특례로 2명 요건 충족**. 세액공제 등 혜택 폭 확대
3. 채용 시 [[킷에이아이-신규채용-시나리오플랜]] + [[5인이하-법인-채용인센티브]] 동시 활용 → 신규 인건비 회수
