> 브랜드 보호 에이전트는 킷에이아이가 **커머스 영역으로 확장**하는 첫 번째 고단가 신상품이 될 수 있다. Enhans의 글로벌 모델을 한국 시장에 맞게 로컬라이징하면, 쿠팡/네이버/11번가 중심의 국내 이커머스 브랜드를 타겟으로 빠르게 시장을 선점할 수 있다.

# 브랜드 보호 에이전트 서비스 설계

## Enhans 원본 모델 요약

### 해결하는 문제
1. 비공식 셀러의 무단 판매
2. 덤핑 가격으로 인한 가격 잠식
3. 채널 간 가격/콘텐츠 혼선
4. 다채널 통합 모니터링 부재

### Enhans 솔루션 흐름
```
모니터링 (1,000+ 플랫폼) → 가격 정책 집행 (MAP 기준) → 자동 위반 대응 (신고/차단)
```

### Enhans 성과
- 비공식 판매자 97% 퇴출
- 영업 이익률 4% 개선
- 집행 조치 6배 증가 (동일 인력)

## 킷에이아이 버전: 한국형 브랜드 보호 에이전트

### 타겟 시장
- **1차 타겟**: 국내 이커머스에서 판매하는 소비재 브랜드
  - 화장품, 건강기능식품, 식품, 패션 브랜드
  - 특히 병행수입/비공식 유통 문제가 심한 K-뷰티, K-푸드
- **2차 타겟**: 해외 진출 한국 브랜드 (역직구)

### 모니터링 대상 플랫폼 (한국 특화)
| 국내 | 해외 |
|---|---|
| 쿠팡 | 아마존 |
| 네이버 스마트스토어 | 쇼피(Shopee) |
| 11번가 | 라쿠텐 |
| G마켓/옥션 | 라자다(Lazada) |
| 카카오 쇼핑 | 알리익스프레스 |
| 위메프/티몬 | 테무(Temu) |

### 서비스 3단계

#### 1단계: 모니터링
- 주요 이커머스 플랫폼에서 브랜드 키워드/SKU 자동 크롤링
- 수집 데이터: 판매자 정보, 판매 가격, 상품 이미지, 상품 설명
- **킷에이아이 기술**: n8n 자동화 + AI 이미지 인식 + 웹 크롤링

#### 2단계: 분석/식별
- 공식 판매자 DB와 대조하여 비공식 판매자 자동 식별
- MAP(최소광고가격) 위반 자동 탐지
- 위조품/병행수입품 이미지 AI 분석
- **킷에이아이 기술**: Claude/GPT 기반 분석 + RAG 공식 판매자 DB

#### 3단계: 대응/보고
- 자동 위반 보고서 생성 (법무팀/영업팀용)
- 플랫폼별 신고 자동화 (쿠팡 신고 API, 네이버 지재권 침해 신고 등)
- 판매자에게 경고/판매 중단 요청 자동 발송
- 주간/월간 대시보드 리포트
- **킷에이아이 기술**: n8n 워크플로우 + AI 보고서 생성 + 이메일 자동화

### 기술 아키텍처

```
[크롤러/모니터링]
    ↓
[데이터 수집 (n8n + 웹 스크래핑)]
    ↓
[AI 분석 엔진 (Claude/GPT)]
  - 판매자 식별
  - 가격 위반 탐지
  - 이미지 분석
    ↓
[공식 판매자 DB (RAG)]
    ↓
[대응 자동화 (n8n)]
  - 보고서 생성
  - 플랫폼 신고
  - 알림 발송
    ↓
[대시보드/리포트]
```

### 가격 체계 (안)

| 플랜 | 월 가격 | 내용 |
|---|---|---|
| Basic | 100만원 | 플랫폼 3개, 주간 리포트, SKU 50개 |
| Pro | 250만원 | 플랫폼 6개, 일간 리포트, SKU 200개, 자동 신고 |
| Enterprise | 별도 협의 | 전 플랫폼, 실시간 모니터링, 맞춤 대응, 전담 매니저 |

### 차별화 포인트 (vs Enhans)

| 구분 | Enhans | 킷에이아이 |
|---|---|---|
| 시장 | 글로벌 50개국 | **한국 시장 특화** |
| 플랫폼 | 1,000+ (글로벌) | **쿠팡/네이버/11번가 심층 분석** |
| 가격대 | 글로벌 기업 대상 (고가) | **중소 브랜드 접근 가능 가격** |
| 부가 서비스 | 없음 | **컨설팅 + 교육 번들** |
| 언어 | 영어 중심 | **한국어 완벽 지원** |
| 대응 | 자동 신고 | **법무 컨설팅 연계 가능** |

### MVP 개발 범위 (Phase 2 실행용)

**MVP 핵심 기능 (4~6주)**
1. 쿠팡 + 네이버 스마트스토어 모니터링
2. 비공식 판매자 자동 식별
3. 가격 위반 자동 탐지
4. 주간 리포트 자동 생성/이메일 발송

**기술 스택**
- 크롤링: Python + Playwright
- 자동화: n8n
- AI 분석: Claude API
- DB: Firebase/Supabase
- 대시보드: Vercel + Next.js
- 알림: Gmail/Slack 연동
