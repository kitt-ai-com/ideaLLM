> [!takeaway] 핵심 테이크어웨이
> TIPS 자격 요건 자체는 충족하지만, **운영사 선투자 2억원 확보가 최대 장벽**이다. 단기적으로 AI바우처/클라우드바우처를 먼저 진행하면서, 중기적으로 TIPS 운영사 네트워킹을 병행하는 **투트랙 전략**이 현실적이다.

## 자격요건 vs 킷에이아이 대조표

| 요건 | TIPS 기준 | 킷에이아이 현황 | 판정 |
|---|---|---|---|
| 업력 | 창업 7년 이내 (신산업 10년) | AI = 신산업, 신생법인 | ✅ 충족 |
| 기술 분야 | 12대 신산업 포함 | AI 에이전트 (LLM+자동화) | ✅ 충족 |
| 팀 구성 | 2인 이상 | 확인 필요 | ⚠️ 확인 |
| 창업팀 지분 | 60% 이상 | 확인 필요 | ⚠️ 확인 |
| VC 지분 | 30% 이하 | 확인 필요 | ⚠️ 확인 |
| **선투자** | **수도권 2억원+ (운영사 투자)** | **미확보 추정** | **❗ 핵심 장벽** |
| 운영사 추천 | 149개 운영사 중 1곳 | 미확보 추정 | ❗ 핵심 장벽 |
| 소재지 | 전국 (수도권/비수도권 차등) | 서울 서초구 (수도권) | ✅ |

### 핵심 판단

```
자격 충족도:  ★★★★☆  (요건 자체는 거의 충족)
실현 난이도:  ★★★★★  (VC 선투자+추천 확보가 핵심)
예상 소요:    3~6개월  (운영사 접촉 → 투자 → 추천 → 접수)
```

**결론:** 자격은 충족하나, TIPS의 본질은 "VC가 먼저 검증한 스타트업에 정부가 매칭 지원"하는 구조. **운영사 선투자 2억원 + 추천 확보**가 사실상의 진입 장벽이다.

## TIPS 프로세스 흐름

```
[1] 운영사 발굴·접촉
     ↓
[2] 운영사 선투자 2억원 확보 + 추천 확약
     ↓
[3] 분기별 정기 접수 (사업계획서 + 추천서)
     ↓
[4] 기술/사업성 평가 (발표 심사)
     ↓
[5] 선정 → R&D 과제 협약 (최대 8억원, 24개월)
     ↓
[6] (성과 우수 시) 딥테크트랙 추가 지원 (최대 15억원)
```

### 운영사 접촉 시 준비 자료

1. **원페이저** (1장): 회사 소개 + 핵심 성과 + 시장 규모 + 요청 사항
2. **피치덱** (10~15장): 문제-솔루션-시장-제품-비즈니스모델-팀-재무-요청
3. **데모**: AI 에이전트 실제 동작 시연 (물류/병원/학원 자동화)
4. **고객 사례**: ROI 수치 포함 도입 사례 2~3건

## 즉시 액션 플랜

### 1단계: 내부 확인 (이번 주)
- [ ] 창업팀 구성원 수 2인 이상 확인
- [ ] 창업팀 지분율 60% 이상 확인
- [ ] 외부 투자자 지분 30% 이하 확인
- [ ] 기존 정부 R&D 수혜 이력 확인

### 2단계: 운영사 접촉 (1~4주)
- [ ] 원페이저 작성
- [ ] 피치덱 10~15장 작성
- [ ] 데모 시연 준비 (n8n+LLM 에이전트)
- [ ] 소풍벤처스, 블루포인트 등 5개사 콜드메일 발송
- [ ] 스타트업 네트워킹 이벤트 참여 (DemoDay, 밋업)

### 3단계: 투자 유치 (1~3개월)
- [ ] IR 미팅 진행
- [ ] 선투자 2억원 텀시트 확보
- [ ] 운영사 추천서 확보

### 4단계: TIPS 접수 (분기별)
- [ ] 2분기 접수 (4~5월): 선투자 확보 완료 시
- [ ] 미확보 시 → 3분기 접수 (7~8월) 준비

## 투트랙 전략: TIPS + 바우처 병행

TIPS 선투자 확보에 3~6개월 소요되므로, 병행 추진:

| 트랙 | 사업 | 시기 | 금액 | 난이도 |
|---|---|---|---|---|
| **단기** (즉시) | 클라우드바우처 | 4/21 마감 | 최대 6,910만원 | ★★ |
| **단기** (하반기) | AI바우처 공급기업 | 하반기 추가모집 | 수요기업당 2억 | ★★★ |
| **단기** (하반기) | 창업성장기술개발 3차 | 하반기 공고 | 최대 2억원 | ★★★ |
| **중기** (3~6개월) | **TIPS** | 2~3분기 | **최대 8억원** | ★★★★★ |

```
[즉시] 클라우드바우처/AI바우처 → 매출+레퍼런스 확보
                ↓
[중기] TIPS 운영사 접촉 → 선투자 확보 → TIPS 선정
                ↓
[장기] 딥테크트랙 추가 (최대 15억) + 후속 VC 투자
```

> **인핸스 성장 공식 참고:** 정부 R&D 4건 → 기술 개발/고객 확보 → 성과 증명 → VC 투자 유치 (누적 290억)
